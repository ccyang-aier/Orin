# 摄影视觉资产规则

摄影教程依赖视觉证据。图片不是装饰，而是第一教学资产：能用画面讲清楚的地方，优先使用真实样张、作品案例、前后对比和拍摄现场信息；当用户没有提供必要图片时，可以充分使用 `imagegen` 生成拟真教学图、器材图、场景模拟、拍摄效果对比和案例模仿。所有图片生成、编辑、改图、局部修改、风格尝试和前后对比默认使用 `imagegen`。

## 1. 资产优先级

1. 用户提供的真实照片或原片；
2. 用户授权使用的作品、样张、截图或前后对比；
3. 可明确来源的公开作品或摄影资料；
4. `imagegen` 拟真教学图，例如不同光线、焦段、景深、快门、色彩和后期效果对比；
5. 教学示意图，例如光线方向、构图骨架、景深关系、站位示意。

生成图可以作为模拟教学案例使用，但必须标注为模拟、生成或示意。不要把生成图写成真实拍摄案例，不要虚构 EXIF、拍摄地点、摄影师或真实作品背景。

## 2. 图像优先工作流

写作或扩写摄影教程时，先做一次图像机会扫描：

- 开篇画面问题是否需要一张失败样张或对比图；
- 核心概念是否需要用同一场景的变量对比说明；
- 拍摄动作是否需要过程图，例如站位、等待、清背景、移动光线或改变焦距；
- 参数、器材或后期步骤是否需要前后对比；
- 练习任务是否需要一组参考样张，帮助读者知道要比较什么；
- 复盘段落是否需要把失败图和改拍图放在一起分析。

除非用户明确要求只停在规划阶段，否则不要只列“样张需求”。如果缺少真实照片，应直接用 `imagegen` 生成合适的模拟教学图，并把图片插入到最靠近相关讲解的位置。每张图附近都要说明：

- 这张图的类型：真实照片、用户照片、公开作品、模拟教学图、生成案例或示意图；
- 它承担的教学任务；
- 读者应该观察哪些画面线索；
- 这张图不能证明什么，例如不能代表真实 EXIF、真实现场或真实摄影师作品。

图片文件位置规则：

- 教程正文使用的图片默认放在当前教程文档所在目录的 `./imgs/` 下；
- Markdown 中默认使用 `./imgs/<filename>` 引用教程图片；
- 只有跨多篇文章复用的全局素材、原始媒体归档或用户明确要求时，才放入仓库级 `media/` 目录；
- 从 `imagegen` 默认输出目录复制项目图片时，最终可引用版本必须落到教程目录的 `./imgs/` 下。

## 3. 真实样张分析

分析真实照片时，尽量记录：

- 主题：这张图主要教什么；
- 可观察线索：光线、主体、背景、构图、色彩、层次、时机；
- 拍摄选择：站位、焦距、光圈、快门、ISO、白平衡、对焦、等待或引导；
- 画面结果：这些选择造成什么视觉效果；
- 可复盘问题：如果重拍，可以改变什么变量。

如果缺少 EXIF 或拍摄信息，不要猜成事实。可以写“从画面推测可能是...”并保持谨慎。

## 4. 前后对比

前后对比适合讲：

- 曝光取舍；
- 构图裁切；
- 主体与背景分离；
- 光线方向变化；
- 色彩统一；
- 后期局部调整；
- 景深和焦距选择。

对比说明必须指出具体差异，不要只说“更好看”。例如：

- 修改后背景亮点被压暗，主体脸部成为最高注意力区域；
- 裁切后右侧边缘杂物消失，视线不再被拉走；
- 色温统一后，肤色和环境灯光不再互相打架。

## 5. imagegen 生成与编辑

默认使用 `imagegen` 处理所有图片类任务，包括：

- 生成拟真相机、镜头、灯具、附件和拍摄现场；
- 生成不同光线、焦段、景深、快门速度、白平衡、曝光取舍的拟真效果对比；
- 生成真实拍摄案例模仿，用于说明街拍、人像、风景、静物、夜景、室内自然光等场景；
- 基于用户图片进行裁切建议、局部修改、风格尝试、前后对比和教学标注方向；
- 生成后期前后对比图、色彩关系图和局部调整示意；
- 生成光线方向、构图骨架、景深关系、站位示意和拍摄流程图。

也可以用 `imagegen` 生成：

- 光线方向示意；
- 主体、背景、相机站位关系；
- 构图骨架和视觉路径；
- 景深区域示意；
- 色彩关系图；
- 拍摄流程图。

生成或编辑要求：

- 图内标签短而清楚；
- 明确它是真实照片、用户照片、模拟教学图、生成案例或示意图中的哪一类；
- 不生成大段文字；
- 生成图可以补足教学样张缺口，但不得伪装成真实纪实证据；
- 不虚构真实 EXIF、摄影师、地点、比赛、展览或作品背景；
- 模仿真实拍摄案例时，描述为“模拟某类拍摄效果”，而不是声称复刻某个真实作品。

常见优先生成的图片类型：

- 同一场景四联图：直接拍、清背景、找光线、等时机；
- 失败类型矩阵：主体不突出、背景太乱、光线太平、颜色偏脏、画面太散、时机错过；
- 参数效果对比：快门速度、光圈、ISO、焦距、白平衡、曝光补偿；
- 拍摄过程图：观察、判断、取舍、表达；
- 后期前后图：原片问题、整体调整、局部引导、最终版本。

## 6. 引用与边界

- 使用外部摄影作品时，记录摄影师、作品名或来源链接；
- 不提供无法确认授权的高清可替代素材；
- 可以用文字分析公开作品，但不要把版权作品重新包装成自己的素材；
- 用户提供图片时，以用户图片为最高优先级，不擅自换成通用素材；
- 如果教程需要样张但当前没有，列出样张需求清单，并说明每张样张承担的教学任务。
