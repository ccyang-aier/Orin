# 摄影视觉资产规则

摄影教程依赖视觉证据。优先使用真实样张、作品案例、前后对比和拍摄现场信息；生成图只做教学示意。

## 1. 资产优先级

1. 用户提供的真实照片或原片；
2. 用户授权使用的作品、样张、截图或前后对比；
3. 可明确来源的公开作品或摄影资料；
4. 教学示意图，例如光线方向、构图骨架、景深关系、站位示意；
5. 生成图片，只能作为示意或视觉草图。

不要把生成图写成真实拍摄案例，不要虚构 EXIF、拍摄地点、摄影师或真实作品背景。

## 2. 真实样张分析

分析真实照片时，尽量记录：

- 主题：这张图主要教什么；
- 可观察线索：光线、主体、背景、构图、色彩、层次、时机；
- 拍摄选择：站位、焦距、光圈、快门、ISO、白平衡、对焦、等待或引导；
- 画面结果：这些选择造成什么视觉效果；
- 可复盘问题：如果重拍，可以改变什么变量。

如果缺少 EXIF 或拍摄信息，不要猜成事实。可以写“从画面推测可能是...”并保持谨慎。

## 3. 前后对比

前后对比适合讲：

- 曝光取舍；
- 构图裁切；
- 主体与背景分离；
- 光线方向变化；
- 色彩统一；
- 后期局部调整；
- 景深和焦距选择。

对比说明必须指出具体差异，不要只说“更好看”。例如：

- 修改后背景亮点被压暗，主体脸部成为最高注意力区域；
- 裁切后右侧边缘杂物消失，视线不再被拉走；
- 色温统一后，肤色和环境灯光不再互相打架。

## 4. 示意图与 imagegen

可以用 `imagegen` 生成：

- 光线方向示意；
- 主体、背景、相机站位关系；
- 构图骨架和视觉路径；
- 景深区域示意；
- 色彩关系图；
- 拍摄流程图。

生成示意图要求：

- 图内标签短而清楚；
- 明确它是示意图，不是真实摄影作品；
- 不生成大段文字；
- 不用生成照片替代真实样张；
- 不用生成图证明某种摄影风格或作品价值。

## 5. 引用与边界

- 使用外部摄影作品时，记录摄影师、作品名或来源链接；
- 不提供无法确认授权的高清可替代素材；
- 可以用文字分析公开作品，但不要把版权作品重新包装成自己的素材；
- 用户提供图片时，以用户图片为最高优先级，不擅自换成通用素材；
- 如果教程需要样张但当前没有，列出样张需求清单，并说明每张样张承担的教学任务。
