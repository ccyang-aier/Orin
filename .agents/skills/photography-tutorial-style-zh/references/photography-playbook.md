# 摄影教程 Playbook

本文件用于写作、修订或扩展中文摄影精品教程。摄影教程的核心不是堆知识点，而是训练读者看见画面、做出拍摄判断、完成练习并复盘。

## 目录

- `1. 教学目标`：摄影精品教程应训练的核心能力；
- `2. 从画面进入`：如何从画面问题而不是抽象定义开场；
- `3. 证据和外部资料`：如何保证摄影知识准确、当前且有依据；
- `4. 技术解释`：曝光、焦距、景深、光线和后期如何连接画面效果；
- `5. 作品分析`：从第一观感走向可迁移拍摄判断；
- `6. 练习设计`：设计可执行、可比较、可复盘的练习；
- `7. 最终自检`：修订前的摄影领域检查清单。

## 1. 教学目标

高价值摄影教程应让读者获得四种能力：

- 观察：看见主体、背景、光线、色彩、层次、边界和时机；
- 判断：知道画面哪里成立、哪里分散、哪里需要取舍；
- 拍摄：能把判断转化为站位、焦距、曝光、构图、等待、引导或后期动作；
- 复盘：能比较不同拍法，说明哪一个更接近创作目标。

避免：

- 只解释参数，不解释画面变化；
- 只说高级、通透、氛围、故事感、电影感，不说明可观察原因；
- 只展示结论图，不展示拍摄选择；
- 把器材升级当作默认答案；
- 把教程写成摄影百科词条。

## 2. 从画面进入

开头优先从读者能感受到的问题或愿望进入：

- 为什么照片看起来平；
- 为什么主体不够突出；
- 为什么背景总是乱；
- 为什么夜景人像容易脏；
- 为什么同一个场景别人拍得更有层次；
- 为什么后期越调越假。

接着把感受翻译成可观察线索：

- 光线方向、硬度、明暗比例；
- 主体与背景的距离、亮度、颜色、形状关系；
- 画面边缘是否干净；
- 视线是否有入口、路径和停留点；
- 色彩是否有主次、呼应和冲突；
- 快门速度是否匹配运动表达；
- 景深是否服务主体，而不是只追求虚化。

## 3. 证据和外部资料

摄影教程需要资料上下文，但资料要服务画面和拍摄判断。

优先资料：

- 摄影书籍、摄影师访谈、展览文字、作品集说明；
- 厂商官方相机/镜头手册、白皮书、样张说明；
- Lightroom、Capture One、Photoshop、手机影像系统等官方文档；
- 可信评测、真实拍摄测试和有方法说明的样张对比；
- 用户提供的原片、EXIF、拍摄现场说明和后期步骤。

使用方式：

- 讲器材能力时，核对厂商或可靠评测，不凭印象写规格；
- 讲软件功能时，核对当前版本能力，避免过时流程；
- 讲摄影师、流派或作品时，尽量记录来源，不编造创作背景；
- 讲画面判断时，把资料变成读者能看见的线索，而不是堆引用；
- 区分事实、画面观察、合理推测和个人审美判断。

避免：

- 把流行说法当成事实；
- 把样张中的 EXIF 缺口补写成确定信息；
- 用器材评测替代摄影教学；
- 为了显得权威而堆摄影师名字、书名或术语；
- 不查资料就评价当前相机、镜头、手机影像或 AI 修图能力。

## 4. 技术解释

技术概念要服务画面。

曝光：

- 不只解释光圈、快门、ISO 的定义；
- 必须说明它们如何影响亮度、运动、景深、噪点和现场取舍；
- 引导读者先判断想保留什么：高光、阴影、肤色、运动轨迹或环境氛围。

焦距与距离：

- 区分焦距带来的视角变化和拍摄距离带来的透视变化；
- 说明广角、标准、长焦如何改变空间关系、背景压缩、主体存在感；
- 避免把某个焦段神化为“人像必备”或“街拍唯一答案”。

景深：

- 同时考虑光圈、焦距、拍摄距离、主体到背景距离和传感器尺寸；
- 把虚化解释为突出主体的一种手段，不是审美目的本身；
- 引导读者比较“虚化更多”和“信息更清楚”之间的取舍。

光线：

- 先观察方向、大小、距离、硬度、颜色和反差；
- 再讨论顺光、侧光、逆光、顶光、窗光、闪光或补光；
- 重点是光线如何塑造体积、情绪、皮肤质感和空间层次。

后期：

- 先确定画面目标，再讲工具步骤；
- 每一步说明解决什么问题，例如曝光平衡、色彩统一、局部引导、肤色保护、背景压暗；
- 避免把调色写成预设参数清单。

## 5. 作品分析

作品分析按照“观感 -> 线索 -> 决策 -> 可迁移练习”展开。

观察顺序：

1. 第一眼看哪里；
2. 视线为什么停在那里；
3. 光线如何组织主体与背景；
4. 构图如何处理边缘、层次和留白；
5. 色彩或黑白关系如何建立情绪；
6. 快门时机和人物姿态是否关键；
7. 这张图的做法能迁移到什么场景。

不要只写：

- 这张照片很高级；
- 这张照片很有故事感；
- 色彩很舒服；
- 构图很稳；
- 情绪很强。

要写成：

- 主体脸部比背景亮半档左右，视线先落到表情；
- 背景中最亮的区域没有贴着主体头部，因此轮廓干净；
- 暖色灯光和冷色阴影形成色温对比，但肤色没有被牺牲；
- 画面左侧留白给人物运动方向，边缘没有抢眼杂物。

## 6. 练习设计

每篇精品教程尽量给练习。练习要可执行、可比较、可复盘。

练习格式：

- 拍摄对象：明确要拍什么；
- 场景限制：光线、时间、地点、器材或焦段；
- 操作变量：一次只改变一两个关键变量；
- 对比方式：至少拍 2-4 张可比较照片；
- 复盘问题：让读者解释差异，而不是只选喜欢哪张。

好练习示例：

- 同一人物、同一背景，分别用顺光、侧光、逆光拍三张，比较脸部立体感和背景分离；
- 同一街角，用 28mm 靠近和 85mm 后退各拍一张，比较空间压缩和现场关系；
- 同一静物，分别保留阴影细节和压低阴影，比较情绪变化。

差练习示例：

- 多拍多练；
- 找一个好看的地方拍照；
- 用大光圈拍出高级感；
- 调出电影感。

## 7. 最终自检

修订摄影教程时，逐项检查：

- 读者是否知道为什么要学这一篇；
- 关键事实、器材能力、软件流程或作品背景是否有可靠资料上下文；
- 每个抽象词是否能落到画面线索；
- 每个参数是否连接画面效果；
- 每个作品判断是否有观察依据；
- 是否出现器材崇拜或万能参数；
- 是否有真实样张、作品案例或明确样张需求；
- 是否有能让读者下一次拍摄就执行的练习。
