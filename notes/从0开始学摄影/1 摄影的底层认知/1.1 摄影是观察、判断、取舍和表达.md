# 1.1 摄影是观察、判断、取舍和表达

![模拟教学图：从观察到表达的拍摄动作流程](<../../../media/从0开始学摄影/1 摄影的底层认知/1.1-observe-decide-select-express.png>)

这是一张模拟教学图，用同一张桌面场景说明拍摄前后的动作变化：先停下来观察现场，再决定主体，接着移除干扰并调整位置，最后让画面在取景器里形成更清楚的表达。它不是真实拍摄作品，也不包含真实 EXIF。

## 1.1.1 这篇先解决什么画面问题

很多新手照片的问题，不是相机不够好，也不是参数不够高级，而是画面里没有一个清楚的拍摄判断。

常见失败会表现为：

1. 主体不明确：照片里有很多东西，但读者不知道第一眼应该看哪里；
2. 背景太抢戏：杂物、亮点、文字、路人或乱线条比主体更显眼；
3. 光线太平或太脏：脸、物体或场景没有体积，暗部混浊，高光刺眼；
4. 画面边缘失控：照片边缘出现半个人、半个物体、突兀亮块或切断的线条；
5. 按快门太早或太晚：人物姿态、街头动作、表情和光影位置没有落在最有意义的一刻；

![模拟教学图：六种新手照片失败类型](<../../../media/从0开始学摄影/1 摄影的底层认知/1.1-beginner-failure-matrix.png>)

这张模拟教学图把六类常见问题放在一起：主体不突出、背景太乱、光线太平、颜色偏脏、画面太散、时机错过。阅读时不要先评价哪张“好不好看”，而要先判断每张图里注意力被什么抢走。

这些问题背后通常不是某一个参数错了，而是拍摄前没有完成四个动作：观察、判断、取舍和表达。

这四个动作可以理解为摄影的最小工作流：

1. 观察：先看见画面里有什么；
2. 判断：再决定什么重要、什么不重要；
3. 取舍：用站位、焦距、曝光、光线、构图和等待删掉干扰；
4. 表达：让画面最终指向一个清楚的感受、信息或关系；

## 1.1.2 拍照前先观察什么

观察不是随便看看，而是把现场拆成几个会影响画面的部分。

### 1.1.2.1 看主体是否真的被看见

主体不是“你想拍的东西”，而是“照片里最先被看见、也最值得停留的东西”。

拍摄前先问：

1. 主体是不是画面里最清楚、最亮、颜色最突出或位置最稳定的部分；
2. 主体周围有没有更亮、更饱和、更清晰、更像人脸或文字的干扰；
3. 主体和背景有没有重叠，例如头顶长出树枝、灯杆穿过身体、招牌贴住脸；
4. 如果把照片缩小到手机屏幕上的小图，主体还能不能被一眼认出来；

如果主体只是出现在照片里，但没有被视觉关系支持，它就很容易被背景吞掉。

### 1.1.2.2 看光线如何塑造体积

光线决定照片是平的，还是有体积和层次。

拍摄前先看：

1. 光从哪里来：顺光、侧光、逆光、顶光或混合光；
2. 光硬不硬：阴影边缘是清楚的，还是柔和过渡的；
3. 主体和背景谁更亮：背景如果比主体亮很多，注意力往往会跑走；
4. 暗部是否脏：阴影里有没有色彩污染、噪点、杂乱纹理或不必要的信息；
5. 高光是否抢眼：白墙、天空、灯箱、金属反光是否比主体更强；

同一个人物站在窗边、室内顶灯下、路灯旁和阴天户外，会得到完全不同的脸部体积、皮肤质感和情绪。

### 1.1.2.3 看背景是否帮主体说话

背景不是空白区域，它会影响读者如何理解主体。

拍摄前先看：

1. 背景有没有和主体发生形状冲突；
2. 背景颜色是否压过主体，或者让主体边界不清；
3. 背景信息是补充主题，还是制造噪音；
4. 主体和背景之间有没有距离，可以通过移动位置或改变焦距拉开层次；
5. 画面边缘有没有不该出现的亮点、文字、半截物体或强线条；

背景干净不等于背景空。好的背景可以说明地点、时间、情绪和人物关系，但它不能抢走主体的第一眼。

### 1.1.2.4 看时机是否已经成立

摄影经常不是找到一个东西就拍，而是等待视觉关系短暂成立。

拍摄前先看：

1. 人物表情有没有到位；
2. 手势、脚步、衣摆、影子和视线方向是否形成更清楚的动作；
3. 路人、车辆、云影、光斑是否会在下一秒进入更好的位置；
4. 主体和背景中的线条、色块、影子是否可能短暂对齐；
5. 当前画面是“差不多能拍”，还是“值得按下快门”；

时机不是只属于街拍和运动摄影。拍静物时，等一束光移动到桌面；拍人像时，等表情放松；拍风景时，等云层露出光线，都是时机。

## 1.1.3 判断：决定照片真正要成立的东西

观察之后要判断。判断的核心不是“这个地方好不好看”，而是“这张照片靠什么成立”。

一张照片通常只能把少数关系讲清楚。可以是：

1. 一个明确主体，例如一张脸、一只手、一个物体、一束光；
2. 一种光线关系，例如侧光让脸部有体积，逆光让轮廓从背景中分离；
3. 一种空间关系，例如前景、中景、背景共同建立层次；
4. 一种色彩关系，例如冷色环境中一个暖色主体被凸显；
5. 一个瞬间动作，例如行人刚好穿过光区，手势刚好和影子重叠；
6. 一种情绪或气氛，例如安静、紧张、孤独、秩序、混乱；

判断越清楚，后面的取舍越容易。判断不清楚时，拍摄者会想把现场所有好看的东西都塞进照片，最后画面变成一张信息拥挤的记录照。

可以用一句话测试自己的判断：

> 这张照片最想让读者看见的是：__________。

如果这句话填不出来，先不要急着按快门。继续观察现场，直到找到主体、光线、空间、色彩或时机里最值得成立的那一项。

## 1.1.4 取舍：摄影不是加法，而是控制注意力

取舍不是后期裁掉一点边角，而是在拍摄现场就开始删掉干扰。

摄影里的取舍可以落到几个具体动作：

1. 改变站位：往左一步避开背景里的灯杆，蹲低一点让主体进入干净天空，靠近一点减少无关环境；
2. 改变距离：靠近可以强化主体，后退可以交代环境，距离变化也会改变透视关系；
3. 改变焦距：广角保留环境关系，长焦压缩背景并简化画面，标准焦距保持自然观察感；
4. 改变曝光：压暗背景亮点，保护天空高光，或者让暗部保留现场氛围；
5. 改变光线：移动主体到窗边、转动身体避开顶光、使用反光板补一点暗部；
6. 改变构图边界：清理画面边缘，让线条、色块和人物位置不互相打架；
7. 等待时机：让路人走出主体背后，让表情放松，让影子移动到更有层次的位置；

取舍的目标不是让画面变得“干净”而已，而是让读者的注意力更稳定地落到你想表达的地方。

## 1.1.5 表达：照片最终要指向什么

表达不是给照片套一个宏大的主题。对新手来说，表达可以非常具体。

例如：

1. 这张照片想表达这个人的安静，所以背景要少，光线要柔，表情要自然；
2. 这张照片想表达街角的忙乱，所以可以保留多层人物和车辆，但主体必须有一个视觉锚点；
3. 这张照片想表达下午阳光的温暖，所以色温、影子方向和高光位置要共同服务这个感受；
4. 这张照片想表达一个物体的质感，所以光线方向、反光控制和背景颜色要突出表面纹理；

表达成立时，读者不一定会说出和拍摄者完全相同的文字，但他的视线会沿着画面提供的线索移动，并停在最重要的位置上。

表达不成立时，读者通常会有这些反应：

1. 不知道该看哪里；
2. 觉得照片有东西，但说不出重点；
3. 注意力被背景亮点、杂物或文字抢走；
4. 感觉画面气氛和主体状态互相矛盾；
5. 觉得照片只是记录了现场，没有形成观看理由；

## 1.1.6 把四个动作放进同一个场景

假设拍摄对象是一杯放在窗边桌面上的咖啡。这个场景看起来简单，但它足够说明观察、判断、取舍和表达。

![模拟教学图：同一场景四张对比](<../../../media/从0开始学摄影/1 摄影的底层认知/1.1-four-panel-comparison.png>)

这张模拟教学图使用同一杯咖啡做四张对比：左一是直接拍，左二是清理背景和边缘，右二是让侧窗光塑造体积，右一是等待光影进入更有表达性的状态。它的作用不是给出标准答案，而是让读者看见：画面变化来自明确的拍摄动作。

### 1.1.6.1 没有判断的拍法

直接站在桌前拍一张，画面里可能同时出现杯子、桌角、手机、纸巾、窗外车辆、墙上插座和桌面反光。

这张照片的问题可能是：

1. 杯子是拍摄对象，但不是最吸引注意力的区域；
2. 窗外高光比杯子更亮，视线被拉走；
3. 桌面杂物让画面显得随意；
4. 杯子和背景颜色接近，轮廓不清楚；
5. 拍摄角度太平，杯子的形状和咖啡表面没有体积；

这不是“咖啡不好看”，而是画面没有完成取舍。

### 1.1.6.2 有判断的拍法

如果判断是“表现下午窗光里一杯咖啡的安静感”，拍摄动作就会变得明确：

1. 观察：找到窗光从侧后方进入的位置，看桌面有没有柔和阴影；
2. 判断：让杯子和杯口反光成为第一注意力，背景只保留一点窗边环境；
3. 取舍：移走手机和纸巾，换一个深色背景方向，避开窗外刺眼高光；
4. 表达：用较低的机位和简洁边缘，让画面更安静、更接近日常观看的距离；

这时，即使没有复杂参数，照片也会更有观看秩序。参数只是帮助完成判断的工具，而不是判断本身。

## 1.1.7 新手失败类型与反推动作

| 失败现象 | 可能缺失的动作 | 下一次改拍动作 |
| --- | --- | --- |
| 主体不突出 | 没有判断第一注意力 | 先说出主体，再移动位置或改变背景，让主体成为最亮、最清楚或最稳定的区域； |
| 背景太乱 | 没有取舍无关信息 | 向左、向右、蹲下或靠近，优先处理主体背后的线条、亮点和文字； |
| 照片太平 | 没有观察光线方向 | 寻找侧光、逆光、窗光或有明暗过渡的位置，让主体出现体积； |
| 颜色很脏 | 没有观察色彩污染 | 避开混合光、彩色墙面反光和杂乱色块，让主体颜色与背景形成主次； |
| 画面太散 | 没有建立表达目标 | 只保留一个主要关系，例如人和光、人和空间、物体和质感、颜色和情绪； |
| 总是差一秒 | 没有等待时机 | 先预判人物、影子、车辆或表情的运动路径，再等关系重合时按快门； |

这个表格的用法不是给照片贴标签，而是帮助复盘：每一次失败都要反推缺了哪个动作。

## 1.1.8 练习任务：同一场景拍四张

这篇的练习不追求拍出漂亮作品，而是训练拍摄前的四个动作。

先参考上面的四联模拟教学图，再用自己的场景拍一组真实练习图。模拟图只负责说明变量，真正的训练来自你亲自拍出的 A、B、C、D 四张对比。

### 1.1.8.1 拍摄对象

选择一个容易控制的对象：

1. 一杯咖啡、一只杯子、一束花、一本书或一个桌面静物；
2. 一个愿意配合的朋友或家人；
3. 一处每天都会经过的门口、窗边、街角或楼道；

不要选择过于复杂的旅行景点或大型活动。第一篇练习的重点是看见变量，而不是处理所有变量。

### 1.1.8.2 拍摄限制

保持以下条件尽量不变：

1. 同一个主体；
2. 同一个大场景；
3. 同一段时间内完成；
4. 不依赖后期大幅修图；

可以使用手机或相机。器材不重要，关键是每张照片必须有明确改变。

### 1.1.8.3 四张照片

拍四张可比较照片：

1. A 图：直接拍，不做整理，作为原始对照；
2. B 图：只改变背景，移动站位或清理干扰，让主体更清楚；
3. C 图：只改变光线，让主体出现更明显的体积、轮廓或影调关系；
4. D 图：只改变时机或表达，等待一个更好的动作、表情、光影位置或画面关系；

每张照片拍完后，不要立刻删除。失败照片也是训练材料。

### 1.1.8.4 合格标准

完成练习后，检查四件事：

1. B 图是否比 A 图更容易看见主体；
2. C 图是否比 A 图更有体积或层次；
3. D 图是否比 A 图更有一个可以被说清楚的观看理由；
4. 四张照片的差异是否来自明确拍摄动作，而不是随机碰运气；

如果四张照片看起来差不多，说明拍摄变量还不够明确。下一次只改变一个更明显的变量，例如换到窗边、压低机位、靠近主体或等待人物走出背景。

### 1.1.8.5 复盘问题

把四张照片放在一起，回答以下问题：

1. 哪一张的主体最清楚，原因是亮度、颜色、清晰度、位置还是背景关系；
2. 哪一张的背景最不抢戏，具体少了什么干扰；
3. 哪一张的光线让主体更有体积，阴影和高光分别在哪里；
4. 哪一张最接近你想表达的感受，它依赖哪些画面线索；
5. 如果只能重拍一次，最值得改变的一个变量是什么；

复盘时不要只写“B 图更好看”。要写成“B 图里主体背后的亮色招牌被避开，杯子轮廓更清楚，所以视线更稳定”。

## 1.1.10 本篇完成标准

读完这一篇后，学习者至少应该能做到：

1. 在按快门前说出照片的主体是什么；
2. 能指出背景中最抢注意力的一个干扰；
3. 能判断现场光线从哪里来，以及它让主体变平还是更有体积；
4. 能用移动、靠近、等待、换背景或改变曝光做出一个明确取舍；
5. 能用一两句话说明照片想表达什么，而不是只说“想拍好看一点”；

摄影学习从这里开始：先训练自己看见画面，再训练相机如何服从判断。
