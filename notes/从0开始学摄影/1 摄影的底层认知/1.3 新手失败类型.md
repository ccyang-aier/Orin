# 新手失败类型

![案例图：新手照片五类失败矩阵](<./imgs/1.3-beginner-failure-types.png>)

这组案例图把五类失败放在同一页：乱、平、脏、散、假。阅读时先不要问“哪张好看”，而要观察每张照片里注意力为什么不稳定：是背景抢走了主体，还是光线没有体积，或者后期让画面失去可信感。

## 1 这篇先解决什么画面问题

新手照片常见的问题，不是“不会用相机”，而是看不出照片为什么失败。

很多照片看起来不成立，通常会落入五种类型：

1. 乱：画面信息太多，主体被背景、边缘、文字、亮点和杂物淹没；
2. 平：主体没有体积，明暗没有层次，空间像被压成一张纸；
3. 脏：颜色、暗部、高光或皮肤质感混浊，画面不清爽；
4. 散：照片里有元素，但它们没有共同指向一个主题或情绪；
5. 假：后期、虚化、磨皮、色彩或光效过度，画面失去真实观看感；

这五个字不是用来批评照片，而是用来诊断照片。只要能把失败类型说清楚，下一次拍摄就能转化成具体动作：换站位、找光线、清背景、等时机、减后期。

## 2 乱：画面里每个东西都在抢话

“乱”的照片最明显的感受是：照片里什么都有，但第一眼不知道看哪里。

### 2.1 先观察什么

判断画面是否乱，可以先看五个位置：

1. 主体背后：有没有树枝、灯杆、招牌、窗框或路人贴着主体；
2. 画面边缘：有没有半截人物、半个杯子、突兀亮块或断裂线条；
3. 最亮区域：照片里最亮的地方是不是主体，还是背景灯箱、天空、白墙；
4. 最强颜色：红色、黄色、高饱和色块是否比主体更吸引眼睛；
5. 文字和人脸：背景文字、人脸和图案是否抢走了阅读优先级；

乱不是元素多。街头、市场、家庭聚会都可以元素很多，但画面必须有一个清楚的视觉锚点。真正的问题是：元素之间没有主次。

### 2.2 常见原因

画面变乱，通常来自这些拍摄选择：

1. 站在原地直接拍，没有绕着主体找更干净的背景；
2. 拍得太远，把无关环境全部收进画面；
3. 没有检查边缘，只盯着主体本身；
4. 背景亮度、颜色或文字比主体更强；
5. 想把现场所有信息都保留下来，结果没有取舍；

### 2.3 现场改拍动作

遇到乱，不要先换器材，先做这些小动作：

1. 向左或向右移动一步，让主体避开背后的杆、树枝和招牌；
2. 靠近主体，减少无关环境，让主体占据更明确的位置；
3. 蹲低或抬高机位，用天空、墙面、桌面或阴影做更简单的背景；
4. 等路人、车辆或影子离开主体背后再拍；
5. 裁掉边缘干扰，但优先在取景时完成，不要完全依赖后期；

![案例图：同一主体从乱到清楚的改拍](<./imgs/1.3-chaos-to-clean-background.png>)

这组对比图的教学任务是看见“清背景”不是把世界清空，而是让主体背后的线条、亮点和颜色不再抢第一眼。

## 3 平：光线和空间都没有起伏

“平”的照片看起来清楚，但没有体积。人脸像贴在背景上，静物像截图，风景像一整块平均亮度。

### 3.1 先观察什么

判断画面是否平，可以看这些线索：

1. 主体有没有高光、阴影和过渡；
2. 主体边缘是否和背景分开；
3. 前景、中景、背景是否有距离感；
4. 画面里是否所有区域都差不多亮；
5. 光线方向是否能被看出来；

如果一张照片只有“曝光正常”，但没有明暗关系、前后关系和边界关系，它往往会显得平。

### 3.2 常见原因

照片变平，常见原因有：

1. 使用正面顺光，主体被均匀照亮，但没有阴影塑形；
2. 阴天或室内顶灯下直接拍，没有寻找侧光或局部光；
3. 主体离背景太近，轮廓被背景吞掉；
4. 拍摄距离和焦距让前后空间被压缩成一层；
5. 后期把高光、阴影和对比度全部拉得过于平均；

### 3.3 现场改拍动作

遇到平，优先让光线和空间有方向：

1. 把主体移到窗边，让侧光在脸部或物体上形成明暗过渡；
2. 让主体离背景远一点，给轮廓留下分离空间；
3. 利用门框、桌面、树影、墙角或道路建立前景到背景的层次；
4. 轻微压暗无关背景，让主体的亮度关系更明确；
5. 等光斑、云影或路灯进入更有方向的位置；

![案例图：平光与侧光的体积对比](<./imgs/1.3-flat-vs-dimensional-light.png>)

这组图用同一人物或静物比较顺光和侧光。重点不是“侧光一定更好”，而是观察阴影如何让脸部、杯口、花瓣或衣褶从背景中立起来。

## 4 脏：颜色和影调让画面不清爽

“脏”的照片不一定是曝光错误。它更像一种混浊感：肤色发灰，暗部发绿，高光刺眼，颜色互相打架，画面不透明。

### 4.1 先观察什么

判断画面是否脏，可以看：

1. 皮肤、白墙、白衣服是否偏绿、偏黄、偏灰或偏红；
2. 暗部里是否有噪点、彩色污染和无意义纹理；
3. 高光是否过曝到失去细节，变成刺眼白块；
4. 画面里是否同时出现多种色温，例如日光、暖灯、霓虹和屏幕光；
5. 后期是否把饱和度、清晰度、HDR 或锐化推得太重；

脏常常来自色彩和光线互相冲突，而不是来自某一个单独参数。

### 4.2 常见原因

照片变脏，常见原因有：

1. 混合光下拍人像，肤色同时被窗光、顶灯和墙面反光影响；
2. 欠曝后强行提亮，暗部噪点和杂色被一起拉出来；
3. 背景颜色太多，主体颜色没有主次；
4. 自动白平衡在复杂光源下摇摆，色温不稳定；
5. 后期为了“有感觉”过度加对比、饱和、暗角和颗粒；

### 4.3 现场改拍动作

遇到脏，先简化光和颜色：

1. 让主体靠近单一光源，例如窗光或一盏稳定灯光；
2. 避开彩色墙面、霓虹灯箱、绿植反光和杂乱 LED 光；
3. 保护肤色和关键高光，宁可背景暗一点，也不要让主体颜色被污染；
4. 拍摄时减少无关高饱和色块，保留一个主色和一个点缀色；
5. 后期先统一白平衡和曝光，再决定是否加风格，不要一开始套重滤镜；

![案例图：混合光导致的脏色与单一光源对比](<./imgs/1.3-dirty-color-vs-clean-light.png>)

这组图用室内人像或桌面静物说明：同一个主体在混合光下容易出现偏色和暗部混浊，换到单一窗光后，颜色关系会更容易控制。

## 5 散：元素存在，但没有共同目标

“散”的照片不像“乱”那样明显拥挤。它可能很干净，却没有力量。主体、背景、光线、色彩和时机各说各话，没有共同指向。

### 5.1 先观察什么

判断画面是否散，可以问：

1. 照片最想让读者看见的对象或关系是什么；
2. 主体、光线、背景和色彩是否都在支持同一个目标；
3. 画面里有没有多个同等重要的视觉中心互相竞争；
4. 空间留白是有情绪，还是只是空；
5. 瞬间是否有动作、表情、光影或位置关系的落点；

散的核心不是东西多，而是没有方向。

### 5.2 常见原因

照片变散，常见原因有：

1. 拍摄前没有一句明确目标，只是觉得现场“还不错”；
2. 主体太小，环境太多，但环境没有提供有效信息；
3. 留白太随意，无法形成安静、孤独、距离或秩序；
4. 色彩、光线和构图分别在表达不同情绪；
5. 时机没有落定，人物姿态和空间关系都处在过渡状态；

### 5.3 现场改拍动作

遇到散，先把表达缩小：

1. 用一句话写下目标：这张照片成立在一个人、一束光、一种颜色、一个动作，还是一个空间关系；
2. 删除不服务这个目标的元素，哪怕它们本身也好看；
3. 让主体和环境发生关系，例如人物看向光、手触碰物体、影子指向主体；
4. 等一个更明确的动作或表情，而不是拍“差不多”的瞬间；
5. 如果环境很重要，就让环境有秩序；如果主体很重要，就让环境后退；

![案例图：从散到集中表达](<./imgs/1.3-scattered-to-focused-expression.png>)

这组图的重点是看见：照片不是因为元素少就集中，而是因为所有画面关系都在支持同一个观看目标。

## 6 假：画面效果超过了可信边界

“假”的照片常见于过度后期、过度美颜、过度 HDR、过度虚化和过度 AI 修图。它可能第一眼很抓人，但看久了会觉得不可信。

### 6.1 先观察什么

判断画面是否假，可以看：

1. 皮肤是否没有毛孔、纹理和自然明暗，只剩塑料感；
2. 虚化边缘是否切到头发、眼镜、手指、树叶或细小物体；
3. 天空、云、光束和反光是否与现场光线方向冲突；
4. 色彩是否过度统一，所有东西都被压成同一种风格；
5. 阴影、高光和清晰度是否不符合真实空间关系；

假不是说照片必须完全纪实。风格化可以成立，但它仍然要尊重画面内部的光线、空间和材质逻辑。

### 6.2 常见原因

照片变假，常见原因有：

1. 先追求“大片感”，没有先判断画面目标；
2. 使用重滤镜掩盖原片的主体、光线和背景问题；
3. 美颜、磨皮和液化破坏了人物真实质感；
4. 手机人像模式或后期景深模拟没有处理好边缘；
5. HDR、清晰度和饱和度过高，让所有细节都在喊；

### 6.3 现场改拍动作

遇到假，不是完全不要后期，而是让后期服从画面：

1. 拍摄时先解决光线和背景，不把全部希望放在修图上；
2. 保留人物皮肤、物体材质和环境光线的自然变化；
3. 使用虚化时检查边缘，宁可少一点虚化，也不要出现错误轮廓；
4. 后期每一步都问：它是在帮助主体，还是在展示效果；
5. 导出前把照片缩小看一遍，检查第一眼是否被滤镜、锐化或假光效抢走；

![案例图：过度后期与可信后期对比](<./imgs/1.3-overprocessed-vs-believable-edit.png>)

这组图用同一张照片比较重滤镜和克制后期。重点不是“原片一定最好”，而是判断调整是否让主体、光线和情绪更清楚。

## 7 一张失败照片的诊断流程

面对一张不满意的照片，不要只说“不好看”。按下面顺序诊断：

1. 第一眼落在哪里；
2. 第一眼是否落在你想要的主体上；
3. 如果不是，被什么抢走了：亮点、颜色、文字、人脸、线条还是边缘杂物；
4. 主体有没有体积：光线方向、阴影、高光和轮廓是否成立；
5. 颜色是否干净：肤色、暗部、高光和背景色是否互相污染；
6. 画面是否集中：所有元素是否服务同一个目标；
7. 后期是否可信：虚化、肤色、色彩、光效和锐化是否超过画面逻辑；

这个顺序可以把“感觉不好”翻译成“下一次要改什么”。

## 8 练习任务：失败照片五类诊断

这篇练习不要求拍新作品，而是先训练诊断能力。

### 8.1 准备材料

从自己的相册里选 10 张不满意的照片，最好包含不同场景：

1. 人像或自拍；
2. 桌面静物或食物；
3. 街头、旅行或日常环境；
4. 夜景或室内灯光；
5. 一张明显修过但仍然不满意的照片；

不要只选最差的照片。那些“好像还行但不耐看”的照片，往往更适合练习。

### 8.2 五类标记

给每张照片标记一个主要失败类型：

1. 乱：主体被背景、边缘、亮点、颜色或文字抢走；
2. 平：光线、空间和明暗关系缺少体积；
3. 脏：颜色、暗部、高光、肤色或质感混浊；
4. 散：主体、背景、光线、色彩和时机没有共同目标；
5. 假：后期、虚化、磨皮、色彩或光效超过可信边界；

一张照片可以同时有多个问题，但练习时先找最致命的一个。真正有用的诊断，必须能反推出一个现场动作。

### 8.3 写出改拍动作

每张照片写三句话：

1. 这张照片最主要的问题是：__________；
2. 我从画面里看到的证据是：__________；
3. 如果现场重拍，我会先做这个动作：__________；

例子：

> 这张照片最主要的问题是乱；证据是人物头部后面有灯箱和文字，画面右边缘还有半截路人；如果重拍，我会向左移动一步，让人物背后变成暗色墙面，并等路人走出画面。

### 8.4 改拍验证

选择其中 2 张可以复拍的场景，按诊断结果重拍：

1. 如果是乱，优先换站位和清边缘；
2. 如果是平，优先找侧光、逆光或前后层次；
3. 如果是脏，优先换到单一光源或减少杂色；
4. 如果是散，优先缩小表达目标；
5. 如果是假，优先减少后期强度，并在拍摄现场解决原始问题；

重拍后把原图和新图并排比较。不要只问新图是否更好看，要问问题是否真的减少了。

### 8.5 复盘问题

完成练习后回答：

1. 我的失败照片最常出现哪一类问题；
2. 这个问题通常发生在什么场景：室内、街头、旅行、人像、食物还是夜景；
3. 我最容易忽视的是背景、光线、颜色、边缘、时机还是后期；
4. 哪一个现场动作最有效：移动一步、靠近、等待、换光、清边缘还是减后期；
5. 下一次拍摄前，我要先检查哪三件事；

## 9 本篇完成标准

读完这一篇后，学习者至少应该能做到：

1. 不再把失败照片笼统归因于器材不好；
2. 能把乱、平、脏、散、假翻译成具体画面线索；
3. 能从失败照片反推出一个现场改拍动作；
4. 能区分“元素多”和“画面乱”、“风格化”和“画面假”；
5. 能用失败照片建立自己的复盘清单；

摄影进步不是从拍出完美照片开始，而是从看懂失败照片开始。看懂一次失败，就多一个下一次能控制的变量。
