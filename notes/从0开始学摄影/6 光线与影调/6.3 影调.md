# 影调

![案例图：高调、低调、中间调、长调和短调](<./imgs/6.3-tonal-keys-and-range.png>)

这组图展示了不同影调的气质：高调轻、亮、柔；低调重、暗、集中；中间调稳定；长调包含从深黑到亮白的宽范围；短调只在较窄亮暗范围内变化，更柔和也更克制。

## 1 这篇先解决什么画面问题

光线是现场条件，影调是照片里的亮暗组织。很多照片问题不是“不够亮”，而是亮暗没有结构。

常见失败包括：

1. 暗部和背景糊在一起，主体没有轮廓；
2. 整张照片都在中间灰，没有重点；
3. 高调照片高光死白，不是轻盈而是丢失；
4. 低调照片黑成一片，不是情绪而是信息缺失；
5. 后期对比拉得太硬，细节和肤色被破坏；

影调要回答：画面里的亮和暗如何分配注意力。

## 2 高调：亮而不空

高调画面整体偏亮，适合轻盈、干净、柔和、纯净的情绪。

关键：

1. 高光要有细节；
2. 主体边界不能消失；
3. 阴影可以很浅，但要有层次；
4. 颜色通常更克制；

高调不是过曝。过曝是信息丢失，高调是亮部组织。

## 3 低调：暗而不死

低调画面整体偏暗，适合安静、戏剧、神秘、压迫或集中注意力。

关键：

1. 主体最重要区域要可读；
2. 暗部可以深，但不能脏；
3. 轮廓光或侧光常能帮助分离；
4. 亮部面积要小而有力；

低调不是欠曝。欠曝是没有信息，低调是有意识地把亮度压低。

## 4 中间调、长调和短调

中间调稳定、自然，适合日常、人像和纪实。它的风险是平，需要用光线方向、色彩或构图建立重点。

长调包含深黑、暗部、中间调、高光和亮白，适合强烈风景、黑白摄影、戏剧场景。它的风险是反差过大，需要保护关键高光和暗部。

短调亮暗范围窄，适合柔和、雾天、阴天、静物和情绪克制的画面。它的风险是缺少视觉锚点，需要形状、色彩或主体姿态支撑。

## 5 影调和光线的关系

同一光线可以被处理成不同影调，同一影调也可以由不同光线实现。

例如：

1. 窗边侧光可以拍成低调人像，也可以补光后拍成中间调；
2. 阴天可以拍成短调柔和街景，也可以后期拉出中高对比；
3. 黄金时刻可以拍成高调逆光，也可以拍成低调剪影；

所以拍摄时既要看光，也要预想最终影调。

## 6 练习任务：同一主体做三种影调

选择同一静物或人像。

### 6.1 拍摄步骤

1. 高调：使用亮背景和柔光，保留细节；
2. 中间调：让主体和背景亮度稳定；
3. 低调：使用暗背景和侧光；
4. 每张都记录最亮区域和最暗区域；

### 6.2 复盘问题

1. 哪张最轻；
2. 哪张最集中；
3. 哪张细节最完整；
4. 哪张只是曝光错误，不是真正影调选择；

## 7 本篇完成标准

完成这一篇后，你应该能做到：

1. 区分高调和过曝、低调和欠曝；
2. 用亮暗分布控制情绪；
3. 在拍摄前预想最终影调；
4. 用影调而不是单纯亮度评价照片；
