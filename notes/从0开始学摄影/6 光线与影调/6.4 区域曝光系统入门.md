# 区域曝光系统入门

![案例图：把亮暗区域当成画面结构](<./imgs/6.4-zone-exposure-tonal-structure.png>)

这组图用灰阶、黑白人像、白布细节、黑色衣物、风景和街道对比说明：区域曝光系统的入门价值不是背数字，而是学会把画面中的黑、灰、白分配到不同角色，让重要信息落在可读的亮暗区域里。

## 1 这篇先解决什么画面问题

很多人看曝光只看整张照片亮不亮，但照片真正成立常常取决于局部亮暗放在什么位置。

常见失败：

1. 白衣服没有纹理；
2. 黑衣服没有层次；
3. 人脸太暗或太亮；
4. 背景亮点比主体更抢；
5. 风景前景黑成死块，天空又白成一片；

区域曝光系统提供的是一种思考方式：每个重要区域应该落在什么亮度位置。

## 2 先观察：画面中的亮暗角色

把画面分成五类就足够入门：

1. 深黑：轮廓、剪影、非常深的阴影；
2. 暗部细节：黑衣服、头发、深色木头、夜景阴影；
3. 中间调：皮肤、灰墙、绿色植物、普通地面；
4. 亮部细节：白衣服、云、浅色墙面；
5. 接近白：灯芯、太阳反光、极亮天空；

不同区域不必都保留完整细节。关键是重要区域不能落错。

## 3 人脸通常需要稳定中间调

在人像里，脸部通常是最重要的信息。它不一定要很亮，但要可读。

观察：

1. 皮肤是否比背景更容易被看见；
2. 脸部高光是否保留质感；
3. 眼窝和鼻影是否太深；
4. 背景亮块是否贴着脸；

如果脸部处在深暗区，照片可能变成剪影或情绪人像；如果不是这个意图，就需要补光、换方向或增加曝光。

## 4 白要有细节，黑要有形状

白色区域不一定必须纯白。婚纱、云、白杯子、白墙和纸张如果是主体，需要保留纹理。黑色区域也不一定必须纯黑。黑衣、头发、暗木头如果重要，需要保留层次。

可以这样判断：

1. 白色主体：略亮，但不能死白；
2. 黑色主体：略暗，但不能糊成一团；
3. 无关高光：可以溢出一点；
4. 无关暗部：可以沉下去；

这就是“信息重要性”决定曝光，而不是颜色本身决定曝光。

## 5 用区域思维安排画面

拍摄前可以做一个简单分配：

1. 主体脸部放在稳定中间调；
2. 白衣服或天空放在亮部但保细节；
3. 暗背景放在深灰或接近黑，帮助主体突出；
4. 小面积高光作为视线锚点；
5. 不重要的边缘暗部可以降低存在感；

这会让曝光、灯光和后期都有方向。

## 6 练习任务：给画面分五个亮暗角色

选择一个高反差室内场景或街景。

### 6.1 拍摄步骤

1. 拍摄前标记深黑、暗部细节、中间调、亮部细节和接近白；
2. 决定主体应该落在哪一类；
3. 拍摄一张保高光版本和一张保主体版本；
4. 后期只做基础曝光和曲线，保持角色清楚；

### 6.2 复盘问题

1. 哪个区域最重要；
2. 哪个区域可以牺牲；
3. 黑色和白色是否有形状；
4. 主体是否落在可读亮度；

## 7 本篇完成标准

完成这一篇后，你应该能做到：

1. 不背复杂区域数字，也能分析亮暗角色；
2. 判断白色和黑色是否需要保留细节；
3. 把主体放在合适亮度层级；
4. 用亮暗结构而不是整体亮度做曝光判断；
