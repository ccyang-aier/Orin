# 人像变形、背景压缩、环境信息与虚化取舍

![案例图：人像中的变形、压缩、环境和虚化取舍](<./imgs/5.5-portrait-distortion-compression-bokeh.png>)

这组图展示了人像拍摄中的几种典型取舍：过近的广角会夸张脸部比例，中等距离更自然，长焦能压缩和简化背景，环境人像需要保留地点信息，过度虚化可能把照片从“人在某处”变成“只有一张脸”。

## 1 这篇先解决什么画面问题

人像最容易被几个简单说法误导：

1. “人像就用长焦”；
2. “背景越虚越高级”；
3. “广角不能拍人”；
4. “焦段决定脸型”；

真正的问题是：你希望这张人像表现人、环境、关系、情绪，还是某种空间距离。

## 2 先观察：人像里环境是不是内容

拍人像前先判断环境角色：

1. 环境是干扰：背景杂乱，应该简化；
2. 环境是身份：咖啡馆、工作室、街道、旅行地说明人物状态；
3. 环境是情绪：窗光、雨夜、阴影、色彩影响气氛；
4. 环境是叙事：人物和地点共同构成故事；

如果环境是内容，虚化过度会损失信息；如果环境只是干扰，简化背景会帮助主体。

## 3 近距离变形：不是广角有罪，是离得太近

广角近拍会让离镜头近的部分变大，例如鼻子、额头、手和肩膀。这种变形有时可以制造亲近和现场感，但也可能让人物显得不自然。

避免不自然变形：

1. 不要把脸贴近镜头；
2. 让人物脸部离画面边缘远一点；
3. 采用中等距离和标准到中长焦；
4. 如果用广角，优先拍环境人像或全身，而不是极近脸部；

广角不是不能拍人，而是需要更精确地控制距离、边缘和姿态。

## 4 背景压缩：长焦让环境靠近主体

长焦后退拍摄时，背景会显得更靠近主体，背景范围变窄，干扰更容易被排除。

适合：

1. 背景有秩序，例如树影、灯光、建筑线条；
2. 想让人物从杂乱环境中跳出来；
3. 希望画面安静、稳定、距离感更强；

风险：

1. 环境信息被删太多；
2. 和被拍者距离变远，互动变弱；
3. 背景压缩后亮点可能贴着头部；
4. 手持稳定和快门速度要求更高；

长焦人像的关键是背景选择，不是只看虚化。

## 5 环境信息与虚化的平衡

虚化是一种注意力控制工具。它可以减少干扰，但也会删除信息。

判断是否虚化过度：

1. 读者还能不能知道人物在哪里；
2. 背景颜色和光线是否还在服务情绪；
3. 主体边缘是否清楚；
4. 画面是否只剩一张漂亮但孤立的脸；

环境人像常需要“可读但不抢戏”的背景。背景不一定清楚到每个细节，但要保留地点、光线和关系。

## 6 人像拍摄的三个常用方案

### 6.1 亲近现场感

1. 较广视角；
2. 中近距离；
3. 保留桌面、街道、房间或动作；
4. 注意边缘和脸部比例；

### 6.2 自然人物关系

1. 标准到中长焦；
2. 舒服沟通距离；
3. 背景有信息但不过度抢；
4. 光线和表情优先；

### 6.3 安静肖像

1. 较长焦距；
2. 后退拍摄；
3. 背景简洁或有规律；
4. 注意眼睛清楚和轮廓分离；

## 7 练习任务：同一个人拍三种人像

找一个愿意配合的人，在同一地点完成三张。

### 7.1 拍摄步骤

1. 广角或较广视角拍环境人像；
2. 标准或中长焦拍自然半身；
3. 长焦后退拍安静肖像；
4. 每张都保留不同程度的背景信息；

### 7.2 复盘问题

1. 哪张最有地点感；
2. 哪张人物比例最自然；
3. 哪张背景最简洁；
4. 哪张虚化让信息损失过多；
5. 被拍者在哪个距离最放松；

## 8 本篇完成标准

完成这一篇后，你应该能做到：

1. 解释人像变形主要来自拍摄距离；
2. 按环境角色选择虚化程度；
3. 用长焦压缩背景时仍然检查背景亮点和轮廓；
4. 不把“人像焦段”和“大光圈”当成唯一答案；
