# 景深变量

![案例图：影响景深的多个变量](<./imgs/5.3-depth-of-field-variables.png>)

这组图用人像和静物说明：景深不只由光圈决定。拍摄距离、焦距、主体和背景距离、背景亮点以及传感器尺寸倾向，都会改变清晰范围和虚化观感。

## 1 这篇先解决什么画面问题

“大光圈虚化”是新手最容易学到、也最容易误用的知识。

常见问题：

1. 背景虚了，但主体表情、光线和构图仍然不好；
2. 人像一只眼清楚，另一只眼不清楚；
3. 多人合影有人虚掉；
4. 静物产品前缘清楚，后缘失焦；
5. 背景没有离主体足够远，即使用大光圈也不够分离；

景深是多个变量共同作用的结果。

## 2 先观察：需要清楚的是一个点，还是一段空间

拍摄前先问：

1. 只需要眼睛清楚，还是脸、手和衣服都要清楚；
2. 静物只要正面清楚，还是整个产品都要清楚；
3. 背景信息是干扰，还是需要交代环境；
4. 主体和背景之间有没有距离；
5. 画面中是否有多个主体处在不同距离；

如果你需要一段空间都清楚，就不能只追求最大光圈。

## 3 光圈：最直接，但不是唯一变量

大光圈让景深变浅，小光圈让景深变深。它是最容易控制的变量。

但要注意：

1. 近距离大光圈的景深会非常薄；
2. 多人合影要考虑每个人是否在同一焦平面；
3. 风景小光圈也要注意衍射和快门速度；
4. 光圈改变同时影响进光，需要配合快门和 ISO；

光圈不是虚化按钮，而是清晰范围按钮。

## 4 焦距和距离：虚化来自空间关系

较长焦距和较近拍摄距离通常更容易让背景虚化。但真正关键的是主体、相机和背景之间的空间关系。

例如：

1. 你靠近主体，景深会变浅；
2. 主体离背景越远，背景越容易虚；
3. 长焦后退能压缩背景，同时让背景更柔；
4. 广角即使开大光圈，如果主体离背景很近，分离仍然有限；

虚化不是只买大光圈镜头。移动主体、改变站位、拉开背景距离，常常更有效。

## 5 传感器尺寸：影响倾向，不决定审美

在相似构图、相似视角和相似光圈下，较大传感器通常更容易获得浅景深；较小传感器更容易让更多区域清楚。

但这不是画质等级排序。手机和小底相机在旅行、纪实、街头和需要全景深的场景里反而更轻松；全画幅在弱光、人像和浅景深控制上更有余量。

传感器只是倾向，最终仍然要看光线、距离、焦距和拍摄目的。

## 6 景深决策

需要浅景深时：

1. 开大光圈；
2. 靠近主体；
3. 让主体远离背景；
4. 选择较长焦距；
5. 找背景亮点和纹理更柔的位置；

需要深景深时：

1. 收小光圈；
2. 适当后退；
3. 避免主体前后距离差太大；
4. 使用三脚架或提高 ISO 支撑快门；
5. 风景中考虑对焦距离和前景；

## 7 练习任务：只改变一个景深变量

选择一盆植物、杯子或人物。

### 7.1 拍摄步骤

1. 固定位置，只改变光圈；
2. 固定光圈，只改变拍摄距离；
3. 固定主体大小，让主体靠近和远离背景；
4. 用广角和长焦各拍一张相似主体大小的照片；

### 7.2 复盘问题

1. 哪个变量让背景变化最大；
2. 哪个变量让主体清晰风险最大；
3. 哪张虚化最多但信息太少；
4. 哪张景深更深但主体仍然突出；

## 8 本篇完成标准

完成这一篇后，你应该能做到：

1. 用五个变量解释景深；
2. 不把虚化当成唯一审美目标；
3. 根据主体数量和空间关系选择清晰范围；
