# 动态范围

![案例图：高反差窗边场景的高光、阴影和后期空间对比](<./imgs/2.2-dynamic-range-highlight-shadow-comparison.png>)

这组图展示同一窗边人物的四种处理：直接按平均亮度拍会让窗外过曝；保护高光会让人物变暗；RAW 后期可以适度拉回阴影；过度恢复暗部会让画面变灰、噪点变多，也会削弱逆光气氛。

## 1 这篇先解决什么画面问题

高反差场景很容易让新手困惑：

1. 拍窗边人物，脸是黑的，窗外是白的；
2. 拍日落，天空很好看，地面变成一片黑；
3. 拍夜景，灯牌有颜色，街道阴影却很脏；
4. 拍白衣服或金属反光，局部亮到没有纹理；
5. 后期把暗部拉亮以后，照片反而灰、脏、假；

这些问题背后都和动态范围有关。

但这一篇不把动态范围讲成器材指标，而是把它放回画面判断：

> 在同一张照片里，你能同时保留多少高光和阴影层次，以及你愿意牺牲哪一边。

动态范围不是为了让所有地方都看清楚。摄影经常需要明暗取舍。真正要学的是：什么时候保护高光，什么时候接受阴影，什么时候用 RAW、HDR 或包围曝光扩展空间，什么时候不要把画面救得太平。

## 2 先观察：高反差画面里谁最容易失控

进入技术之前，先看画面。

高反差场景里通常有三类区域：

1. 关键高光：天空、窗外、白衣服、皮肤反光、灯牌文字、金属高光；
2. 关键阴影：人脸暗部、黑色衣服、室内角落、背光建筑、树影里的主体；
3. 可牺牲区域：无意义黑角、刺眼灯泡中心、深色背景、过曝但不重要的反光；

拍摄前先问：

1. 这张照片最不能丢的是高光纹理，还是主体暗部表情；
2. 高光是画面信息，还是只是一块干扰亮斑；
3. 阴影是需要看清的主体，还是应该保留的气氛；
4. 如果把暗部全部拉亮，照片会更清楚，还是更平；
5. 如果让高光白掉，画面会失去关键内容，还是仍然成立；

动态范围的学习从这里开始：不是追求全部细节，而是判断哪些细节值得被保留。

## 3 动态范围：一张照片能容纳的明暗距离

动态范围可以理解为相机从最亮到最暗还能记录层次的范围。

画面里反差越大，越容易出现两种失败：

1. 高光溢出：亮部变成没有纹理的白，例如天空没有云、白衣服没有褶皱、灯牌文字糊成一片；
2. 阴影堵死：暗部变成没有层次的黑，例如人脸看不清、黑衣服和背景混在一起、室内角落没有结构；

动态范围越宽，设备越有可能同时保留更多亮部和暗部信息。但它依然不能替你决定：照片应该亮在哪里，暗在哪里。

### 3.1 高光不是越亮越好

高光是照片里最容易抢走注意力的区域。亮部如果没有意义，它会把视线从主体身上拉走。

需要特别保护的高光包括：

1. 有云层、颜色或形状的天空；
2. 人脸、皮肤、白衣服和婚纱上的细节；
3. 玻璃、金属、陶瓷等有质感的反光；
4. 夜景灯牌上的文字和颜色；
5. 食物、饮品、静物表面的亮部纹理；

可以适度牺牲的高光包括：

1. 灯泡中心；
2. 太阳本体；
3. 画面边缘的小反光；
4. 不承担信息的纯白窗外；
5. 不影响主体的远处亮点；

保护高光的关键不是让照片变暗，而是让关键亮部仍然有可读纹理。

### 3.2 阴影不是越清楚越好

很多后期失败来自一个误解：暗部都应该被拉亮。

阴影有时是信息，有时是气氛。

需要保留细节的阴影包括：

1. 主体脸部暗侧；
2. 黑色衣服边缘；
3. 室内人物动作；
4. 背光建筑的结构；
5. 静物暗部的轮廓和材质；

可以保持深暗的阴影包括：

1. 不重要的背景角落；
2. 用来分离主体的暗背景；
3. 强化情绪的低调区域；
4. 画面边缘的无意义杂物；
5. 为了突出高光而压住的暗部；

阴影全被拉亮以后，照片可能更“看得清”，但也可能失去方向、层次和情绪。

## 4 曝光取舍：先决定保护谁

高反差场景拍摄时，最重要的不是找到一个万能曝光，而是明确优先级。

![案例图：同一逆光场景的三种曝光取舍](<./imgs/2.2-exposure-choice-backlit-scene.png>)

这组图把同一逆光人物拆成三种结果：按人物脸部曝光，表情清楚但窗外丢失；按高光曝光，窗外层次保住但人物变暗；折中曝光保留一定高光，再用后期把关键阴影抬回一点。观察重点不是哪张绝对正确，而是哪张更符合照片目标。

### 4.1 保护高光

当高光承担重要信息时，优先保护高光。

适合保护高光的场景：

1. 日出、日落、云层和雪景；
2. 白色衣服、婚纱、浅色花瓣；
3. 玻璃杯、金属、陶瓷、皮肤油光；
4. 夜景灯牌、霓虹和舞台灯；
5. 窗外风景是画面的一部分；

拍摄动作：

1. 观察画面里最重要的亮部；
2. 降低曝光，让亮部仍然有纹理；
3. 检查主体是否还能通过后期、补光或站位救回；
4. 如果主体太暗，优先考虑移动主体或改变拍摄方向，而不是硬拉阴影；

保护高光会让照片变暗一点，这是正常的。关键是暗部能否服务气氛，或者能否在 RAW 后期里适度恢复。

### 4.2 保护主体暗部

当主体表情、动作或信息比高光更重要时，可以优先保护主体暗部。

适合保护主体暗部的场景：

1. 背景窗外只是无意义白光；
2. 人物表情比天空重要；
3. 室内纪实里动作发生在阴影中；
4. 画面本来就想表达明亮、轻快、生活化的观感；
5. 高光区域很小，过曝不会破坏主体；

拍摄动作：

1. 让主体脸部或关键物体达到可读亮度；
2. 接受不重要高光的损失；
3. 避免皮肤或白衣服被一起过曝；
4. 必要时通过换角度避开刺眼高光；

这种选择不是技术失败，而是表达优先级不同。

### 4.3 保留反差

有些照片的魅力正来自强烈明暗。比如舞台侧光、街头光斑、窗边剪影、夜晚霓虹、低调静物。

这类照片不需要把所有阴影拉亮。更好的做法是：

1. 保留主体轮廓；
2. 让高光有方向；
3. 暗部只恢复关键边缘；
4. 保持明暗之间的戏剧性；

如果把阴影全部打开，强光和暗影之间的关系会变平，照片会像一张被过度解释的记录图。

## 5 RAW 的潜力：它能救回余量，不能创造不存在的信息

RAW 在高反差场景里非常有用，因为它通常保留了更多亮部、暗部和白平衡调整空间。

适合用 RAW 的高反差场景：

1. 窗边人像；
2. 逆光街拍；
3. 日出日落；
4. 夜景灯牌；
5. 雪地、海边、沙漠等大面积亮部场景；
6. 舞台、展览、室内混合光；

RAW 后期常见动作：

1. 适度降低高光，让天空、白衣服或灯牌纹理回来；
2. 适度提高阴影，让主体暗部有可读细节；
3. 调整白平衡，让肤色和环境光不互相打架；
4. 用局部调整引导注意力，而不是全局把所有暗部拉亮；
5. 控制噪点和清晰度，避免暗部变脏；

但 RAW 不是后悔药。

如果高光已经彻底过曝，画面里没有记录纹理，后期很难真正恢复；如果阴影严重欠曝，强行提亮会出现噪点、色块和灰雾。RAW 给的是余量，不是凭空创造细节。

## 6 HDR：把多次判断合成一张更宽的画面

HDR 的意思可以先简单理解为：用多张不同曝光的照片，合成一张同时保留更多高光和阴影的照片。

很多手机会自动做 HDR：拍摄时快速记录亮、中、暗几张，再合成一张高光不过曝、暗部也更清楚的照片。

HDR 适合：

1. 风景里的天空和地面反差很大；
2. 室内空间需要同时看清窗外和房间；
3. 建筑、房产、城市景观这类强调信息完整的题材；
4. 手机拍摄高反差场景，希望直出更稳；

HDR 不适合或需要谨慎的情况：

1. 人物、树叶、车辆、水面等快速运动，容易出现重影；
2. 画面本来需要强反差和剪影；
3. 灯光氛围依赖深阴影；
4. 合成过度后，照片变得没有主次，局部像塑料；

HDR 的问题不在于技术本身，而在于它容易让人误以为“所有地方都亮起来”就是好照片。

## 7 包围曝光：给高反差场景留保险

包围曝光是指在同一构图下拍几张不同亮度的照片，例如一张偏暗、一张正常、一张偏亮。

它的价值是：当你不确定该保护高光还是阴影时，先把选择保留下来。

适合包围曝光的场景：

1. 日出日落风景；
2. 室内窗景；
3. 建筑室内外同时重要；
4. 不会快速移动的静物和城市景观；
5. 重要拍摄，现场不能重来；

拍摄动作：

1. 固定构图，最好使用三脚架或稳定支撑；
2. 拍一张保护高光的暗曝光；
3. 拍一张中间曝光；
4. 拍一张保护阴影的亮曝光；
5. 回到电脑或手机后比较，选择单张最佳，或用于 HDR 合成；

包围曝光不是每次都要合成。很多时候，你只是需要从几张里选出最符合表达目标的一张。

## 8 什么时候不要过度恢复

动态范围越强，越容易诱惑你把所有东西都拉回来。但摄影不是监控画面，不需要所有区域都同等清楚。

不要过度恢复的情况：

1. 阴影本来用于隐藏杂乱背景；
2. 画面情绪依赖低调和神秘感；
3. 逆光轮廓比脸部细节更重要；
4. 夜景灯光需要深色环境衬托；
5. 暗部恢复后出现明显噪点、脏色和灰雾；
6. 高光恢复后变成灰白色，失去光感；

判断是否过度恢复，可以问：

1. 主体是否更清楚，还是背景也一起抢戏了；
2. 明暗方向是否还存在；
3. 高光看起来像发光，还是像灰色涂块；
4. 阴影是否保留了情绪，还是被拉成无意义细节；
5. 读者第一眼是否更稳定，还是到处都一样亮；

好的后期不是把直方图填满，而是让画面目标更明确。

## 9 拍摄决策：高反差现场的最小流程

遇到高反差场景，可以按这个顺序处理：

1. 找主体：这张照片最重要的是谁；
2. 找关键高光：天空、窗外、白衣服、灯牌、皮肤反光是否重要；
3. 找关键阴影：主体暗部、动作、轮廓和环境信息是否需要看清；
4. 决定优先级：保高光、保主体，还是保留强反差；
5. 选择格式：重要场景优先 RAW，快速分享可用 JPEG 或手机 HDR；
6. 选择曝光：让关键区域不丢，接受不重要区域的牺牲；
7. 必要时补救：移动位置、改变方向、用反光板或灯补一点暗部；
8. 复盘后期：只恢复支持主体和表达的区域；

这个流程比记住“欠曝几档”更有用，因为不同场景的答案不一样。

## 10 练习任务：手机和相机拍同一高反差场景

这也是本章练习。目标是比较手机和相机在高反差场景中的直出观感、暗部细节、高光保护和后期空间。

### 10.1 拍摄对象

选择一个稳定的高反差场景：

1. 窗边人物或静物；
2. 室内看向窗外；
3. 日落时的街角；
4. 夜景灯牌前的人或物；
5. 阳光下的白墙、树影和人物；

不要选变化太快的场景。练习重点是比较动态范围和曝光取舍，不是抓拍。

### 10.2 拍摄方法

至少拍 6 张：

1. 手机普通模式直出；
2. 手机 HDR 或自动夜景模式直出；
3. 相机 JPEG，按主体曝光；
4. 相机 JPEG，降低曝光保护高光；
5. 相机 RAW，降低曝光保护高光；
6. 如果相机支持，拍一组包围曝光；

保持构图尽量一致。每张照片记录一句当时的曝光目标，例如“保人物脸”“保窗外云”“保灯牌文字”“接受背景变黑”。

### 10.3 复盘问题

对比照片时，不要只选“最好看”的一张。逐项回答：

1. 哪一张高光保留最好，保住了什么信息；
2. 哪一张暗部最干净，暗部里有没有噪点或脏色；
3. 哪一张主体最清楚，但牺牲了哪些亮部；
4. 手机 HDR 有没有让画面更稳，是否也让反差变平；
5. 相机 RAW 能恢复哪些区域，哪些区域恢复不了；
6. 包围曝光里哪一张单张最接近你的表达目标；
7. 如果合成 HDR，画面有没有失去光线方向；
8. 哪些阴影其实不该被拉亮；
9. 下一次遇到类似场景，你会先保护高光还是主体暗部；

## 11 本篇记住这几句话

1. 动态范围不是为了让所有地方都清楚，而是给高光和阴影取舍更多空间；
2. 高光丢失通常比阴影变暗更难补救，但不是所有高光都重要；
3. 阴影可以是信息，也可以是气氛，不必全部拉亮；
4. RAW 能保留更多后期余量，但不能创造没有记录下来的细节；
5. HDR 和包围曝光适合信息完整的高反差场景，但可能削弱明暗情绪；
6. 高反差照片的核心问题不是技术指标，而是你愿意让读者看见什么、看不见什么；
