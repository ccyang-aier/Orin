# 相机、传感器、镜头、焦段、RAW与JPEG

![案例图：同一场景在手机、相机、广角、长焦、RAW 和 JPEG 下的画面差异](<./imgs/2.1-camera-sensor-lens-raw-jpeg-comparison.png>)

这组对比图用于观察同一扇窗边人物在不同工具选择下的变化：手机直出更方便，较大传感器更容易保留暗部层次；广角交代环境但边缘更容易变形，长焦简化背景并压缩空间；RAW 版本后期空间更大，JPEG 版本则更像已经被相机预先做完判断。

## 1 这篇先解决什么画面问题

很多人一开始学摄影，会把工具理解成参数清单：相机是什么品牌，传感器多大，镜头多少毫米，RAW 文件有多大，JPEG 颜色好不好。

但在真正拍照时，更重要的问题是：

1. 为什么同一个人用手机拍显得背景很乱，用相机加长焦拍就更突出；
2. 为什么同一个咖啡杯，广角靠近会有现场感，长焦后退会更安静；
3. 为什么一张照片天空白成一片，另一张还能拉回云层；
4. 为什么 JPEG 看起来马上能发，RAW 却更适合认真修；
5. 为什么换了设备以后，照片不一定更好，只是给了你不同的取舍空间；

这一篇不把工具写成百科，而只回答一个问题：

> 它会怎样改变画面？

如果一个工具概念不能帮助你看见主体、背景、空间、光线、色彩或后期取舍，它暂时就不是最小必要知识。

## 2 先观察：工具差异最终会落在画面哪里

拿到一张照片时，先不要急着猜器材。先看画面结果：

1. 主体和背景有没有分开；
2. 空间是显得宽、近、夸张，还是显得窄、远、压缩；
3. 暗部有没有细节，还是只剩一团黑；
4. 高光有没有层次，还是白成一片；
5. 边缘人物或建筑有没有明显拉伸；
6. 色彩和锐化是否已经很重，后期再调是否容易变脏；
7. 画面噪点是破坏细节，还是还能接受；

这些才是工具知识真正要服务的观察点。

例如拍一个站在窗边的人，背景里有书架、灯、墙面和窗外亮光。你真正要判断的不是“我需要多贵的相机”，而是：

1. 主体脸部是否能从背景里分离；
2. 窗外高光是否需要保留；
3. 背景信息是帮助交代环境，还是抢走注意力；
4. 这张照片是要快速记录，还是要认真后期；

工具不是替你完成判断，而是让某些判断更容易实现。

## 3 相机：它不是答案，而是控制入口

相机最重要的作用，不是让照片自动变高级，而是提供更稳定的控制入口。

你可以把相机理解成三个能力的集合：

1. 取景能力：你能不能清楚看到画面边缘、主体位置、背景干扰和光线变化；
2. 控制能力：你能不能主动决定曝光、对焦、焦距、景深、快门和文件格式；
3. 记录能力：它能不能把现场的亮部、暗部、颜色和细节保存下来，留给后期继续取舍；

手机、卡片机、微单、单反和中画幅不是从低级到高级的单线排名。它们更像不同拍摄方式：

| 工具选择 | 画面优势 | 常见代价 |
| --- | --- | --- |
| 手机 | 随手、计算摄影强、直出稳定、适合快速记录和分享； | 真实焦距很短，空间关系容易偏广角，复杂高反差和自然虚化空间有限； |
| 小型相机 | 轻便，控制比手机更直接，适合日常观察训练； | 镜头和传感器空间有限，极端弱光或大幅后期余地有限； |
| 可换镜头相机 | 焦段、景深、对焦、RAW 和附件选择更丰富； | 需要更多拍摄判断，器材变重，错误也会被更清楚地记录下来； |

选择工具时先问画面目标：

1. 我要的是快速抓到瞬间，还是慢慢组织画面；
2. 我要保留现场环境，还是压缩背景突出主体；
3. 我要直接得到可用颜色，还是要保留后期空间；
4. 这张照片最怕的是高光丢失、暗部太脏、背景太乱，还是时机错过；

相机不会替你决定照片成立在哪里。它只是让你更明确地处理这些问题。

## 4 传感器：影响画面的光线余量和景深关系

传感器是接收光的平面。对初学者来说，不必先背各种尺寸名词，先理解它会影响三个画面结果：

1. 弱光画质：光线少时，画面是否容易出现粗糙噪点和脏暗部；
2. 动态范围：高光和阴影是否能同时保留更多层次；
3. 景深倾向：在相似构图和视角下，较大传感器更容易得到浅景深，较小传感器更容易让更多区域清楚；

### 4.1 传感器影响暗部是否干净

在室内、夜晚、阴影里拍摄时，照片常见问题不是“不够亮”这么简单，而是暗部变得脏：

1. 阴影里出现彩色噪点；
2. 人脸暗部失去细节；
3. 黑色衣服和背景糊成一团；
4. 后期提亮以后，画面像蒙了一层灰；

更大的传感器通常给你更多弱光余量，但它不是魔法。光线方向不好、曝光取舍错误、背景太乱时，较大的传感器也只能记录一张更清楚的失败照片。

拍摄时更有用的动作是：

1. 先让主体靠近可用光源，例如窗边、门口、灯下侧光；
2. 观察主体脸部和背景谁更亮；
3. 必要时宁可保留暗部气氛，也不要把所有阴影都硬拉亮；
4. 如果现场反差很大，优先保护关键高光，再判断暗部是否有必要救回；

### 4.2 传感器影响背景虚化的难易

很多人把较大传感器等同于“背景更虚”。更准确地说，在相似视角、相似构图和相似光圈条件下，较大传感器更容易得到浅景深。

但背景虚化不是照片变好的保证。它只解决一个问题：减少背景信息，让主体更突出。

它不能解决：

1. 主体表情不好；
2. 光线方向太平；
3. 背景亮点仍然刺眼；
4. 构图边缘混乱；
5. 画面没有表达目标；

所以判断传感器和景深时，先问：

> 背景信息是需要被删除，还是应该被保留来说明环境？

人像、静物和杂乱街景里，浅景深可以帮助主体分离；旅行、纪实、建筑和环境人像里，过度虚化可能会把有用信息一起删掉。

## 5 镜头：它改变你和世界的观看关系

镜头不是只负责“清不清楚”。它真正改变的是视角、空间、背景和主体存在感。

观察一支镜头对画面的影响，可以看四件事：

1. 它能看到多宽；
2. 它会让你站得多近或多远；
3. 它会怎样改变主体和背景的比例；
4. 它会怎样处理清晰区域和模糊区域；

镜头选择会迫使你改变身体位置。身体位置一变，主体、背景、边缘和空间关系都会变。

## 6 焦段：不是数字，而是空间取舍

焦段最容易被讲成数字，但学习时更应该把它翻译成画面感。

![案例图：广角、标准、长焦拍摄同一人物时的空间关系](<./imgs/2.1-focal-length-space-comparison.png>)

这组图用同一人物和同一街角做对比：广角靠近时环境进入画面，主体和背景距离被拉开，边缘更容易夸张；标准焦段接近日常观看，人物和环境关系比较自然；长焦后退时背景被压缩，杂乱信息被简化，主体更容易从环境中跳出来。

### 6.1 广角：把环境带进画面

广角适合交代空间，但它也会放大现场里的混乱。

广角画面的常见特点：

1. 能拍进更多环境，适合街头、室内、旅行、建筑和环境人像；
2. 靠近主体时，前景会变大，空间会更有冲击；
3. 边缘更容易拉伸，人物放在边缘时可能变形；
4. 背景信息很多，主体如果没有亮度、颜色或位置优势，很容易被吞掉；

广角拍摄时最重要的不是“拍得多”，而是控制边缘和前景。你需要更主动地处理：

1. 画面四角有没有半截杂物；
2. 主体是否离边缘太近；
3. 前景是否有意义；
4. 线条是否把视线引向主体，而不是把画面切乱；

### 6.2 标准焦段：接近日常观看

标准焦段的优势是自然。它不会像广角那样夸张空间，也不会像长焦那样强烈压缩背景。

它适合训练基础判断：

1. 主体和背景是否分开；
2. 站位是否合适；
3. 光线是否帮主体塑形；
4. 边缘是否干净；
5. 画面是否有一个清楚的观看理由；

标准焦段不容易靠视觉夸张救场。它会逼你认真观察现场关系。

### 6.3 长焦：简化背景，压缩空间

长焦常被用来突出主体，因为它会让视角变窄，并且在你后退拍摄时让背景看起来更集中、更接近主体。

长焦画面的常见特点：

1. 背景范围变小，杂乱信息更容易被排除；
2. 主体和背景的比例关系更稳定，空间有压缩感；
3. 更容易形成浅景深，让主体从背景中分离；
4. 拍摄距离变远，现场互动感可能降低；

长焦适合人像、舞台、体育、街头局部、远处光影和需要简化背景的场景。它的误区是：只追求虚化，而忽略环境关系。背景太虚时，照片可能干净了，但场景也变得没有信息。

## 7 RAW 与 JPEG：一个保留余地，一个完成预判

RAW 和 JPEG 不是谁高级谁低级，而是两种不同的工作方式。

可以这样理解：

1. RAW 更像保存了更多原始信息，适合后期继续决定曝光、白平衡、色彩和局部层次；
2. JPEG 更像相机已经帮你做了一次压缩、锐化、降噪、色彩和对比度判断，适合快速得到稳定结果；

### 7.1 RAW 改变的是后期取舍空间

RAW 的价值不在于让照片自动好看，而在于当画面里有高光、阴影、色温和色彩冲突时，你还有更多调整余地。

适合拍 RAW 的情况：

1. 逆光人像，需要同时照顾肤色和背景高光；
2. 日出、日落、窗边、雪地、舞台灯这类反差大的场景；
3. 需要认真调色、统一肤色或做局部引导；
4. 重要照片，不希望被直出风格提前锁死；
5. 白平衡不确定，例如室内混合光或夜景霓虹；

但 RAW 也不是万能。过曝到没有信息的白色区域，后期很难凭空恢复；严重欠曝的阴影强行提亮，也可能带来噪点和脏色。

### 7.2 JPEG 改变的是拍摄现场的判断压力

JPEG 的优势是快、轻、稳定。相机或手机会把锐化、降噪、颜色、对比和局部亮度处理提前做好。

适合拍 JPEG 的情况：

1. 日常记录和即时分享；
2. 光线简单、反差不极端的场景；
3. 活动、旅行、家庭照片等需要快速交付的内容；
4. 你已经喜欢相机或手机直出的颜色；
5. 后期需求很少，只需要轻微裁切和亮度调整；

JPEG 的代价是后期空间较小。大幅拉高暗部、恢复高光或改变白平衡时，画面更容易断层、偏色或变脏。

所以 JPEG 拍摄更要求现场判断：曝光、白平衡和直出风格尽量当场确定，不要把所有问题都留给后期。

## 8 拍摄决策：按画面目标选择工具

不要先问“我该买什么”，先问“我这张照片要解决什么画面问题”。

| 画面目标 | 优先考虑的工具选择 | 现场动作 |
| --- | --- | --- |
| 主体从杂乱背景中分离 | 长一点的焦段、主体离背景远一点、较大光圈或较大传感器余量； | 后退取景，避开背景亮点，让主体处在更干净的亮暗关系里； |
| 交代人物和环境关系 | 广角或标准焦段，保留足够环境信息； | 靠近主体但控制边缘，让环境服务人物而不是淹没人物； |
| 高反差场景要认真后期 | RAW，必要时降低曝光保护高光； | 先看天空、窗外、灯箱和皮肤高光，再决定曝光补偿； |
| 快速记录生活瞬间 | 手机或 JPEG，选择稳定直出风格； | 优先抓住时机和表情，不为复杂设置错过画面； |
| 夜晚或室内保持暗部干净 | 可用光源、更稳的持机、更合适的快门和较好高 ISO 余量； | 让主体靠近光，避免后期硬拉无意义暗部； |

器材选择永远和画面取舍绑定。换工具之前，先把问题说清楚：是主体问题、背景问题、空间问题、光线问题，还是后期空间问题。

## 9 练习任务：同一场景拍出工具差异

这次练习的目标不是证明哪个设备更好，而是看见工具如何改变画面。

### 9.1 拍摄对象

选择一个高可控场景：

1. 窗边人物；
2. 桌面静物；
3. 街角店铺前的人；
4. 有前景、中景、背景的走廊或楼梯；

要求画面里至少包含主体、背景和一个明显亮部，例如窗户、天空、灯牌或反光桌面。

### 9.2 拍摄方法

拍 4 到 6 张可比较照片：

1. 手机或相机 JPEG 直出一张；
2. 如果有相机，用 RAW 再拍一张相同构图；
3. 用较广视角拍一张，保留环境；
4. 用较长焦段或后退取景拍一张，简化背景；
5. 如果能控制光圈，拍一张背景更清楚的版本；
6. 再拍一张背景更虚、主体更突出的版本；

每次只改变一个主要变量。不要一边换焦段，一边换角度，一边换光线，否则复盘时很难判断差异来自哪里。

### 9.3 复盘问题

把照片放在一起比较：

1. 哪一张主体最容易被看见，原因是亮度、颜色、清晰度、背景简化，还是空间压缩；
2. 哪一张最能交代环境，哪一张最简洁；
3. 哪一张边缘最混乱，为什么；
4. 哪一张暗部最干净，高光最完整；
5. JPEG 版本有没有已经把颜色和对比做得太重；
6. RAW 版本后期能救回什么，又不能救回什么；
7. 如果只能保留一张，你保留的是工具效果，还是画面判断；

## 10 本篇记住这几句话

1. 相机不是答案，而是观察、控制和记录的入口；
2. 传感器影响光线余量、动态范围和景深倾向，但不能替代画面判断；
3. 镜头和焦段改变的不是数字，而是你和主体、背景、空间之间的关系；
4. 广角让环境进入照片，长焦帮助简化背景，标准焦段训练基础观察；
5. RAW 保留后期取舍空间，JPEG 提前完成很多直出判断；
6. 工具知识只有落到主体、背景、光线、空间、色彩和后期余地上，才真正有用；
