# 曝光与曝光意图

曝光不是让照片“正常亮”，而是决定一张照片要保留什么、牺牲什么、让读者先看见什么。相机测光只能给出一个技术建议，拍摄者必须把建议翻译成画面意图。

本篇围绕六个问题展开：

1. 曝光不是拍亮，而是决定保留什么；
2. 光圈、快门、ISO 如何改变画面；
3. 测光、曝光补偿、直方图和高光警告如何帮助现场判断；
4. 什么时候保高光、保暗部、向右曝光，什么时候保持现场反差；
5. A/Av、S/Tv、M 模式如何映射到不同操作方式；
6. 运动、弱光和高反差里如何做曝光取舍；

## 2 本篇完成标准

完成本篇后，学习者应该能做到：

1. 拍摄前先说出这张照片最想保留的亮部、暗部、肤色、动作或氛围；
2. 解释光圈、快门和 ISO 各自改变的不是参数，而是景深、运动、噪点和亮度余量；
3. 用曝光补偿表达意图，而不是盲目相信测光零点；
4. 用直方图和高光警告判断是否需要重拍；
5. 在 A/Av、S/Tv、M 中选择最适合当前场景的操作方式；
6. 在弱光、运动和高反差中说明自己牺牲了什么；

## 3 本篇练习总任务

选择同一高反差场景，例如窗边人物、日落风景、夜景灯牌或白色杯子与深色背景。拍摄三组：

1. 保高光：让窗外、天空、白色物体或灯光保留细节；
2. 保主体：让人物脸部、静物纹理或暗部主体有足够可读信息；
3. 保氛围：允许暗部沉下去，不把现场拉成均匀明亮；

复盘时对每张照片写一句话：

> 这张曝光保留了__________，牺牲了__________，因此画面更接近__________。
