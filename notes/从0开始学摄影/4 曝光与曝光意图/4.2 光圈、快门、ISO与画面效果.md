# 光圈、快门、ISO与画面效果

![案例图：光圈、快门和 ISO 的画面效果](<./imgs/4.2-aperture-shutter-iso-effects.png>)

这组图把参数变化翻译成可见结果：大光圈让背景更虚，小光圈让更多区域清楚；高速快门凝固水花，慢速快门拉出运动；低 ISO 暗部更干净，高 ISO 弱光下能拍到但噪点更明显。

## 1 这篇先解决什么画面问题

光圈、快门和 ISO 常被讲成曝光三角，但新手真正需要的是把它们和画面问题对应起来。

常见困惑包括：

1. 为什么照片亮度差不多，背景虚化却完全不同；
2. 为什么画面不暗，人物动作还是糊的；
3. 为什么夜景提亮后暗部很脏；
4. 为什么大光圈人像眼睛清楚但耳朵和另一个人不清楚；
5. 为什么 ISO 高了照片能拍到，但质感下降；

三个参数都影响亮度，但它们更重要的作用不是亮度，而是画面性格。

## 2 先观察：这张照片最怕哪种失败

调整参数前先判断最怕什么：

1. 最怕背景抢主体：优先考虑光圈、焦距、距离和主体背景距离；
2. 最怕动作糊：优先考虑快门速度；
3. 最怕暗部脏：优先考虑光线、曝光和 ISO；
4. 最怕多人不清楚：优先考虑景深，而不是一味大光圈；
5. 最怕高光丢失：优先考虑曝光补偿和动态范围；

参数不是独立按钮，它们服务具体失败类型。

## 3 光圈：控制进光，也控制景深

光圈越大，进入镜头的光越多，景深通常越浅；光圈越小，进入的光越少，景深通常越深。

大光圈适合：

1. 背景杂乱，需要突出主体；
2. 弱光环境，需要更多进光；
3. 人像、静物、细节，需要柔化背景；

大光圈的代价：

1. 对焦容错低，近距离时尤其明显；
2. 多人合影容易有人不清楚；
3. 背景虚化不能解决光线平、表情差、色彩脏；

小光圈适合风景、建筑、静物整体、多人合影和需要前后都清楚的场景。但小光圈会减少进光，可能需要更慢快门、更高 ISO 或三脚架。

## 4 快门：控制时间，也控制运动表达

快门速度决定相机记录多长时间的光。时间越短，越容易凝固动作；时间越长，越容易产生拖影。

高速快门适合：

1. 儿童、动物、运动、街拍瞬间；
2. 水花、衣摆、头发、手势等短暂动作；
3. 长焦手持时降低抖动风险；

慢速快门适合：

1. 水流、云、车灯、人流的轨迹；
2. 夜景和低光静态场景；
3. 有意表现速度和流动；

注意：防抖只帮助抵消手持抖动，不会让运动主体停止。主体在动，就需要快门速度或有意接受运动模糊。

## 5 ISO：提高感光放大，也放大代价

ISO 可以理解为对信号的放大。ISO 提高后，画面更容易达到需要的亮度或快门速度，但暗部噪点、色彩纯度和细节会受到影响。

使用 ISO 时要看：

1. 当前光线是否足够；
2. 快门速度是否必须提高；
3. 暗部是否是画面重点；
4. 后期是否还需要大幅提亮；

不要为了低 ISO 让快门过慢，也不要为了拍亮把 ISO 拉到超出可接受范围。照片糊掉通常比噪点更难救，但噪点也会破坏暗部和肤色。

## 6 三个参数不是固定答案，而是取舍组合

同一亮度可以由不同组合得到：

1. 大光圈、低 ISO、较快快门：背景虚、画面干净，适合人像；
2. 小光圈、低 ISO、慢快门：前后清楚，需要三脚架或静态场景；
3. 中等光圈、高 ISO、快快门：动作清楚，但噪点增加；
4. 小光圈、高速快门、高 ISO：风景和运动同时要求高时，画质压力最大；

每次调整一个参数，都要问它对景深、运动和噪点产生了什么代价。

## 7 练习任务：固定亮度，改变画面效果

选择一个人物或静物，背景保持不变，再准备一个有运动的对象，例如倒水、走动或挥手。

### 7.1 拍摄步骤

1. 同一主体，用大光圈和小光圈各拍一张；
2. 同一运动，用高速快门和慢速快门各拍一张；
3. 同一弱光角落，用低 ISO 慢快门和高 ISO 快快门各拍一张；
4. 尽量让照片亮度接近，只比较画面效果；

### 7.2 复盘问题

1. 哪个参数改变了背景；
2. 哪个参数改变了动作；
3. 哪个参数改变了暗部质感；
4. 哪一张牺牲最值得，哪一张牺牲最明显；

## 8 本篇完成标准

完成这一篇后，你应该能做到：

1. 把光圈连接到景深和主体突出；
2. 把快门连接到运动凝固或拖影；
3. 把 ISO 连接到弱光、快门和噪点；
4. 不再只按亮度理解曝光三要素；
