# 测光、曝光补偿、直方图、高光警告

![案例图：测光、曝光补偿、直方图和高光警告](<./imgs/4.3-metering-histogram-highlight-warning.png>)

这组图展示了相机取景界面、曝光刻度、直方图和高光提示。它们不是要替你决定照片，而是提醒你当前记录是否接近意图，哪些区域可能已经丢失信息。

## 1 这篇先解决什么画面问题

相机会测光，但相机不知道你想表达什么。

常见失败包括：

1. 雪景被相机压成灰色；
2. 黑色背景被相机提亮，低调氛围消失；
3. 逆光人像脸部太暗；
4. 白色衣服或天空过曝，回不来细节；
5. 夜景灯牌局部死白，画面注意力被亮块抢走；

测光给的是“相机认为平均合理”的建议。摄影师要决定这个建议是否符合画面意图。

## 2 先观察：相机最容易被什么骗

测光容易被这些情况影响：

1. 大面积白色：雪地、白墙、婚纱、天空，相机会倾向压暗；
2. 大面积黑色：夜景、黑衣、暗背景，相机会倾向提亮；
3. 强逆光：背景亮度远高于主体，相机会难以兼顾；
4. 小面积强光：灯、反光、太阳，会造成局部溢出；
5. 主体不在中央：某些测光逻辑可能偏向画面整体而非主体；

看到这些情况，就不要盲目相信曝光刻度中间。

## 3 测光模式：相机从哪里读亮度

常见测光可以这样理解：

1. 评价/矩阵测光：综合全画面，适合大多数练习；
2. 中央重点：偏向画面中央，适合主体在中央附近；
3. 局部测光：关注较小范围，适合主体和背景反差较大时；
4. 点测光：只读很小区域，精确但容易误用；

新手不用一开始频繁切测光模式。更重要的是理解：无论什么测光，最终都要用曝光补偿和回放检查来确认。

## 4 曝光补偿：把相机建议推向你的意图

曝光补偿不是修错，而是表达意图。

常见使用：

1. 雪地、白墙、明亮天空：适当增加曝光，让白保持白；
2. 黑背景、低调人像、夜景：适当减少曝光，让暗保持暗；
3. 逆光人像：根据脸部和背景取舍，可能需要增加曝光或补光；
4. 日落剪影：减少曝光，让轮廓和天空更清楚；

曝光补偿的关键是边看边调。不要背固定数值。每个场景的主体、背景和光线比例都不同。

## 5 直方图：看明暗信息有没有贴边

直方图显示画面亮度分布。它不是好照片评分表，只是信息分布图。

读直方图时看：

1. 是否大量贴右边：高光可能溢出；
2. 是否大量贴左边：暗部可能堵死；
3. 分布是否符合场景：夜景偏左、雪景偏右都可能正常；
4. 是否有单个颜色通道溢出：红花、霓虹、舞台灯容易出现；

不要为了让直方图居中而毁掉画面。直方图服务意图，不替代意图。

## 6 高光警告：提醒你哪里可能回不来

高光警告通常用闪烁或标记提示过亮区域。它特别适合检查：

1. 天空云层；
2. 白衣服、婚纱、白色产品；
3. 人脸高光和额头反光；
4. 夜景灯、金属反光、水面反光；

如果高光区域是太阳或灯芯，溢出可能可以接受；如果是人物脸部、衣服纹理或云层细节，通常需要重新曝光或改变拍摄方式。

## 7 现场最小判断流程

拍摄高反差场景时，可以按这个顺序：

1. 先决定主体和关键高光；
2. 用评价测光得到初始曝光；
3. 根据画面意图做曝光补偿；
4. 拍一张测试；
5. 回放看直方图和高光警告；
6. 放大看主体关键位置；
7. 需要时重拍；

这个流程比反复猜参数更可靠。

## 8 练习任务：让测光犯错，再修正它

准备三种场景：白色物体、黑色背景、逆光人物或静物。

### 8.1 拍摄步骤

1. 每个场景先按相机建议拍一张；
2. 再分别用正负曝光补偿拍两张；
3. 回放查看直方图和高光警告；
4. 写下哪一张最符合你的画面意图；

### 8.2 复盘问题

1. 相机在哪个场景最容易误判；
2. 曝光补偿改变的是亮度，还是画面重点；
3. 哪些高光溢出可以接受，哪些不能；
4. 直方图是否帮助你发现肉眼忽略的问题；

## 9 本篇完成标准

完成这一篇后，你应该能做到：

1. 理解测光只是建议；
2. 用曝光补偿表达画面意图；
3. 用直方图判断明暗是否贴边；
4. 用高光警告决定是否需要重拍；
