# 曝光不是拍亮，而是决定保留什么

![案例图：同一窗边场景的不同曝光意图](<./imgs/4.1-exposure-as-intention.png>)

这组图展示同一窗边人物的不同曝光结果：有的保住窗外高光，有的让脸部更明亮，有的保留低调气氛，也有明显过亮和过暗的失败版本。阅读时不要先问哪张“正确”，先问每张照片决定保留什么。

## 1 这篇先解决什么画面问题

很多新手把曝光理解成“照片要亮一点”。于是拍到逆光、夜景、室内窗边和日落时，会本能地把所有地方都拉亮。

结果常见问题是：

1. 窗外天空白成一片，照片失去环境；
2. 人脸虽然亮了，但现场的暗部气氛消失；
3. 夜景被拍得像白天，灯光不再有情绪；
4. 为了救暗部强行提高 ISO，画面噪点和脏色明显；
5. 后期把阴影拉开，照片变灰、变平、变假；

曝光不是让全部区域都看清，而是在有限动态范围里决定谁最重要。

## 2 先观察：画面里谁不能丢

按快门前先找四类区域：

1. 关键高光：天空、窗户、灯、白衣服、金属反光、水面反光；
2. 关键暗部：人物脸部阴影、黑色衣服、前景轮廓、室内环境；
3. 情绪暗部：可以保留黑、深、安静、神秘或压迫感的区域；
4. 无关区域：可以过亮、过暗或不清楚，但不会破坏主题的区域；

然后问：

> 如果只能保留一个区域，这张照片最不能丢的是哪里？

这句话比测光零点更重要。

## 3 保高光：让亮部有形状

保高光不是把照片拍暗，而是让亮部保留可读信息。

适合保高光的场景：

1. 日落、云层、雪地、白墙、婚纱、窗外景色；
2. 灯箱、霓虹、舞台灯和城市夜景；
3. 产品、静物和金属反光；
4. 需要保留环境时间和天气的风景；

保高光的代价通常是暗部变深。这个代价可以接受吗，要看暗部是不是主体。如果暗部只是轮廓或气氛，变深可能更好；如果暗部是人物脸部，就可能需要补光、换站位或后期恢复。

## 4 保主体：让真正重要的地方可读

保主体通常发生在人像、纪实、静物和产品里。你希望读者先看到脸、动作、手、物体纹理或关键细节。

保主体时要注意：

1. 主体亮了，背景高光可能会溢出；
2. 主体如果靠近窗光或反光面，曝光更容易稳定；
3. 如果背景比主体亮很多，只调曝光可能不够，需要换角度或补光；
4. 肤色比背景更重要时，可以牺牲一部分窗外细节；

曝光不能单独解决所有主体问题。主体太暗时，除了加曝光，还可以让主体靠近光源、转身体、换背景、使用反光板或等待光线变化。

## 5 保氛围：暗不是错误，暗部也可以表达

低调画面、夜景、窗边侧光和舞台摄影里，暗部常常是情绪的一部分。如果把所有阴影都拉亮，画面反而失去现场感。

保氛围时要分清：

1. 有意的暗：暗部有形状、有层次、服务情绪；
2. 失败的暗：主体消失，读者不知道看哪里；
3. 干净的暗：阴影颜色稳定，噪点可接受；
4. 脏暗部：色彩污染、噪点和压缩痕迹明显；

暗不是问题，不知道为什么暗才是问题。

## 6 一张照片的曝光句子

拍摄前可以用一句话确定曝光意图：

> 我希望这张照片保留__________，可以牺牲__________，因为我想让画面呈现__________。

示例：

1. 保留脸部和眼神，可以牺牲窗外细节，因为这是一张人物状态照片；
2. 保留云层高光，可以牺牲前景细节，因为这是一张日落风景；
3. 保留夜晚灯光和阴影，可以牺牲暗部完整细节，因为这张照片要有夜的气氛；

这句话越清楚，曝光补偿、测光、RAW 后期和补光选择越容易。

## 7 练习任务：同一场景拍三种曝光意图

选择一个窗边人物或静物，背景中必须有明显亮区。

### 7.1 拍摄步骤

1. 第一张：保高光，让窗外或亮部不明显溢出；
2. 第二张：保主体，让人物脸部或静物纹理可读；
3. 第三张：保氛围，让暗部保持低调，不强行拉亮；
4. 每张都回放查看高光和暗部；

### 7.2 复盘问题

1. 哪张最接近真实现场；
2. 哪张最清楚说明主体；
3. 哪张最有情绪；
4. 哪张的牺牲最不能接受，为什么；

## 8 本篇完成标准

完成这一篇后，你应该能做到：

1. 不再把曝光等同于变亮；
2. 拍摄前说出最想保留的区域；
3. 接受高光、暗部和主体之间的取舍；
4. 用一句曝光意图指导下一次拍摄；
