# 0 课程总纲

## 0.1 教学定位

《从0开始学摄影》是一套面向零基础到进阶学习者的摄影技术教程。它不把摄影理解为器材参数的堆叠，也不把审美写成空泛感受，而是训练学习者持续建立四种能力：看见画面、判断取舍、完成拍摄、复盘改进。

本教程的核心问题是：一张照片为什么在视觉上成立，为什么不成立，以及下一次拍摄时如何用具体动作让它更成立。

课程主线如下：

1. 看见画面：识别主体、光线、背景、空间、色彩、时机和情绪；
2. 控制工具：理解相机、曝光、对焦、焦距、景深和菜单配置如何服务拍摄判断；
3. 建立画面秩序：用光线、影调、空间、色彩、视觉重量和构图控制注意力；
4. 完成工作流：从导入、选片、后期到输出，建立完整的作品生产流程；
5. 进入真实场景：在静物、日常、人像、风景、夜景、街拍和旅行中综合运用；
6. 形成创作系统：通过作品分析、审美训练、长期项目和输出机制建立个人表达。

## 0.2 如何使用本教程

本教程可以按三种方式学习：

1. 线性学习：从第1章读到第12章，适合完全从零开始建立完整体系；
2. 按问题跳读：根据当前拍摄问题进入对应章节，例如曝光不稳读第4章，主体不突出读第7章；
3. 按场景强化：已经有基础的学习者可以直接进入第10章，再回到前文补齐技术短板。

每篇正式教程都应尽量遵循同一套结构：

1. 画面问题：先指出读者能看见的失败现象或创作目标；
2. 观察前置：在讲技术前，先说明拍摄前应该看什么、等什么、比较什么；
3. 核心概念：解释必要的摄影知识，但必须连接画面效果；
4. 拍摄决策：把概念转换成站位、焦距、曝光、对焦、光线、构图或后期动作；
5. 练习任务：给出可以执行、可以比较、可以复盘的拍摄练习；
6. 复盘问题：让学习者说明差异原因，而不是只判断喜欢哪一张。

## 0.3 练习与复盘模板

每章至少设计一个练习，且练习不能只写“拍什么”。它必须包含感知训练，让观察力、预判力和等待能力成为显性训练。

练习模板：

1. 观察任务：拍摄前先观察光线方向、背景干扰、人物运动路径、色彩冲突或画面边缘；
2. 拍摄任务：明确要拍摄的对象、场景和限制条件；
3. 变量说明：一次只改变一到两个关键变量，例如光线方向、焦距、站位、快门速度或背景距离；
4. 合格标准：说明怎样才算完成练习，例如主体更清楚、背景更干净、影调更稳定、视线更集中；
5. 失败反思：记录失败原因，例如等待不足、边缘失控、曝光意图不清、对焦模式错误或后期目标模糊。

摄影日志模板：

1. 拍摄目标：这次想解决什么画面问题；
2. 现场观察：光线、主体、背景、空间、色彩、时机分别是什么状态；
3. 拍摄选择：使用了什么焦距、曝光、对焦、站位、构图和等待策略；
4. 样张对比：至少选择两到四张照片比较差异；
5. 下一次实验：只选择一个最重要的问题，在下一次拍摄中验证。

## 0.4 文档拆分规则

本教程采用“每章一个导读文档，每个小节一个教程文档”的组织方式。章节文档负责建立地图，小节文档负责完成真正的教学闭环。

不建议把一整章写成一篇长教程。每章通常包含多个不同的画面问题和训练目标，如果全部写进一篇文档，容易变成知识堆叠，也不利于 Obsidian 检索、引用、复盘和后续扩写。

推荐目录结构：

```text
notes/从0开始学摄影/
  0 课程总纲.md
  1 摄影的底层认知/
    1 摄影的底层认知.md
    1.1 摄影是观察、判断、取舍和表达.md
    1.2 好照片的基础.md
    1.3 新手失败类型.md
    1.4 手机计算摄影与相机光学摄影.md
  2 工具的最小必要认识/
    2 工具的最小必要认识.md
    2.1 相机、传感器、镜头、焦段、RAW与JPEG.md
```

文档颗粒度规则：

1. 总纲文档：负责全局导航、课程设计原则、章节顺序和写作标准；
2. 章节导读文档：负责说明本章学习目标、前置能力、知识地图、练习总任务和本章完成标准；
3. 小节教程文档：负责完整展开一个具体知识点或画面问题，必须包含画面问题、观察前置、核心概念、拍摄决策、练习任务和复盘问题；
4. 大型小节可以继续拆分，例如“对焦配置入门”后续可以拆成对焦区域、Servo与One Shot、主体识别、追焦失败原因；
5. 拆分标准不是字数，而是教学闭环：如果一个主题已经有独立的画面问题、操作方法、练习和复盘标准，就应独立成篇；
6. 第一阶段默认按“小节等于一篇教程”推进，等某个小节内容膨胀到多个独立问题时再继续拆分。

## 0.5 课程大纲

### 1 摄影的底层认知：从按快门到看见画面

本章建立摄影学习的总入口：摄影不是先学器材，而是先学会看见画面中的问题。

1.1 摄影是观察、判断、取舍和表达；

1.2 好照片的基础：主体、光线、背景、空间、色彩、时机、情绪；

1.3 新手失败类型：乱、平、脏、散、假；

1.4 手机计算摄影与相机光学摄影：为什么相机不一定直出更好看；

本章练习重点：从自己已有照片中挑出五张失败照片，分别标注主体、光线、背景、空间、色彩和时机的问题。

### 2 工具的最小必要认识

本章只建立必要工具概念，避免学习者过早进入器材清单。所有工具知识都要回答一个问题：它如何改变画面。

2.1 相机、传感器、镜头、焦段、RAW/JPEG；

2.2 动态范围：RAW潜力、HDR、包围曝光、高反差场景；

2.3 附件只讲能改变画面的工具：三脚架、CPL、ND、反光板；

2.4 数据安全底线：存储卡、备份、文件命名；

本章练习重点：用手机和相机拍摄同一高反差场景，比较直出观感、暗部细节、高光保护和后期空间。

### 3 相机操作系统：以 Canon EOS R6 Mark II 为示例

本章不是复刻说明书，而是建立“按钮、菜单和自定义配置如何服务拍摄判断”的操作系统。正文写作时以通用相机逻辑为主，以 Canon EOS R6 Mark II 作为具体示例。

3.1 拿起相机前：电池、存储卡、镜头、肩带、固件、时间与版权信息；

3.2 机身按钮地图：快门、主拨轮、速控拨轮、方向摇杆、AF-ON、Q、MENU、回放、录像、模式转盘；

3.3 拍摄界面读图：取景器和屏幕里每个关键图标代表什么；

3.4 Q 快速菜单、主菜单、My Menu 的分工；

3.5 核心菜单配置：画质、RAW/JPEG、白平衡、测光、驱动、快门类型、防抖、触控、显示、回放；

3.6 对焦配置入门：AF区域、人物/动物/车辆识别、Servo、One Shot、追踪灵敏度；

3.7 自定义按钮与 C1/C2/C3：把常用判断放到手指能立刻触达的位置；

3.8 新机推荐初始设置：适合新手的稳妥默认值，以及哪些设置不要急着改；

本章练习重点：不用离开取景器，完成曝光补偿、切换对焦区域、改变驱动模式、放大回看检查清晰度。

### 4 曝光与曝光意图

本章把曝光从“照片亮不亮”提升为“我要保留什么”。曝光是画面意图，而不是相机读数。

4.1 曝光不是拍亮，而是决定保留什么；

4.2 光圈、快门、ISO 与画面效果；

4.3 测光、曝光补偿、直方图、高光警告；

4.4 保高光、保暗部、ETTR、就地还原；

4.5 A/Av、S/Tv、M：理解曝光后的操作映射；

4.6 运动、弱光、高反差中的曝光取舍；

本章练习重点：在同一高反差场景中分别保高光、保暗部和向右曝光，比较RAW后期恢复空间与画面情绪。

### 5 焦距、距离、景深与空间感

本章建立镜头如何构建空间的统一模型。焦距、距离和光圈不是三个孤立知识点，而是共同决定透视、空间压缩、背景关系和景深。

5.1 焦距改变视角，距离改变透视；

5.2 广角、标准、长焦的空间性格；

5.3 景深变量：光圈、焦距、距离、背景距离、传感器尺寸；

5.4 二维画面中的三维感：近大远小、遮挡、重叠、大气透视、色彩纵深；

5.5 人像变形、背景压缩、环境信息与虚化取舍；

本章练习重点：用广角靠近、标准焦距正常距离、长焦后退三种方式拍同一主体，比较空间关系和主体存在感。

### 6 光线与影调

本章把光线和影调区分开：光线是现场条件和塑形手段，影调是画面亮暗区域的整体分布关系。两者共同决定照片的体积、情绪和层次。

6.1 观察光：方向、硬度、大小、距离、色温、反差；

6.2 自然光：窗光、阴天、黄金时刻、蓝调时刻、夜景灯光；

6.3 影调：高调、低调、中间调、长调、短调；

6.4 区域曝光系统入门：把亮暗分布当成画面结构；

6.5 人工光入门：光质控制、单灯逻辑、反光板、TTL/手动、混合光；

6.6 扩展方向：双灯、三灯、棚拍、外拍、高速同步；

本章练习重点：同一主体分别用顺光、侧光、逆光拍摄，再分别处理成高调和低调，比较体积、情绪和层次变化。

### 7 色彩、视觉重量与构图：如何控制注意力

本章的核心逻辑是：色彩产生视觉重量，视觉重量决定注意力分配，注意力分配就是构图。本章不是把色彩和构图并列摆放，而是从颜色关系走向画面组织。

7.1 色彩基础：冷暖、互补、相近、主色、点缀色；

7.2 拍摄现场的色彩控制：背景、服装、环境反光、肤色污染；

7.3 视觉重量：高光、饱和色、人脸、文字、清晰区域；

7.4 注意力分配：主体为什么被看见，干扰为什么抢戏；

7.5 格式塔原理：接近、相似、连续、闭合；

7.6 构图方法：三分法、中心构图、引导线、框中框、边缘管理；

本章练习重点：在同一场景中分别用色彩、亮度、边缘管理和线条引导突出主体，比较哪一种最稳定。

### 8 工作流、选片与后期思维

本章先讲判断，再讲软件。后期不是从滑块开始，而是从“这张RAW需要被修成什么”开始。

8.1 导入、筛选、命名、归档、备份；

8.2 接触表与选片：粗筛、细筛、终选；

8.3 用选片反推拍摄问题：重复、犹豫、错过、边缘失控；

8.4 后期前先判断：这张RAW需要被修成什么；

8.5 无损编辑、整体优先、局部其次、风格最后；

8.6 Lightroom、Capture One、Photoshop 的角色边界；

本章练习重点：从一次拍摄的至少三十张照片中完成三轮选片，并写出终选照片胜出的具体原因。

### 9 后期与输出

本章把后期工具连接到画面目标。每一步调整都必须说明它解决了什么问题，例如曝光平衡、肤色保护、背景压暗、色彩统一或视线引导。

9.1 基础调整：曝光、对比、白平衡、曲线、高光、阴影；

9.2 局部调整：主体、背景、天空、肤色、视线引导；

9.3 色彩调整：HSL、色调统一、黑白转换、避免脏色；

9.4 AI工具：降噪、智能蒙版、内容识别修复、生成式修改边界；

9.5 色彩管理：sRGB、P3、AdobeRGB、显示器、导出；

9.6 打印与输出：分辨率、纸张、色域、成册、展览；

本章练习重点：同一张RAW分别修成自然版、情绪版和黑白版，说明每个版本的画面目标和取舍。

### 10 场景实战：从可控到复杂

本章按照变量复杂度递进：先从静物这种最可控场景开始，再进入日常、人像、风景、夜景、街拍和旅行。

10.1 通用检查清单：找光、清背景、定主体、控影调、等时机；

10.2 静物与产品：形状、质感、反光、单灯控制；

10.3 日常记录：普通生活中的秩序、细节、情绪；

10.4 人像摄影：光线、背景、姿态、表情、如何与被拍者工作；

10.5 风景摄影：时间、天气、前景、尺度、动态范围；

10.6 夜景与弱光：稳定、噪点、色彩污染、环境氛围；

10.7 街拍：距离、等待、层次、预判、伦理与沟通；

10.8 旅行摄影：地点感、人物、细节、叙事序列；

本章练习重点：每个场景都先完成观察前置，再拍摄三到五张可比较照片，并写出现场取舍。

### 11 作品分析与审美训练：摄影师如何看世界

本章是输入系统，目标是帮助学习者从外部世界提取可迁移判断。这里训练的是如何看作品、看场景、看失败，而不是急着定义个人风格。

11.1 作品分析路径：观感、线索、决策、迁移；

11.2 大师作品不只看风格，要看拍摄选择；

11.3 影像叙事基础：单张、组照、序列、节奏；

11.4 摄影类型的观看逻辑：纪实、街头、人像、风景、观念摄影；

11.5 失败照片分析：哪里不成立，下一次如何改；

11.6 建立参考库：光线、影调、空间、色彩、构图、题材；

11.7 每周看图训练：三张作品、三个判断、一个可迁移练习；

本章练习重点：选择三张不同摄影类型的作品，分别写出第一眼、视觉线索、拍摄决策和可迁移练习。

### 12 高阶创作与长期输出：摄影师如何建立自己的世界

本章是输出系统，目标是帮助学习者形成自己的主题、立场和作品组织方式。前一章训练如何看世界，本章训练如何建立自己的世界。

12.1 摄影项目意识：主题、边界、对象、时间跨度；

12.2 个人命题：我为什么拍、长期关心什么、愿意反复回到哪里；

12.3 长期拍摄方法：样本积累、重复观察、阶段复盘；

12.4 项目编辑：选片、排序、节奏、删减、标题与说明文字；

12.5 个人风格：稳定的观看方式，而不是滤镜；

12.6 摄影伦理：肖像权、公共空间、真实边界、AI修改边界；

12.7 输出系统：发布、成册、展览、作品集、长期复盘；

本章练习重点：设计一个可以持续四周的小型摄影项目，明确主题、边界、拍摄对象、选片标准和最终输出形式。

## 0.6 写作标准

后续拆分正式教程时，必须遵守以下标准：

1. 每篇教程都从具体画面问题进入，而不是从抽象定义进入；
2. 每篇教程都必须包含观察前置任务，例如先观察光线方向、背景干扰、人物运动路径、色彩冲突，再开始讲技术；
3. 练习不能只写拍什么，必须写拍之前看什么、等待什么、比较什么；
4. 第7章必须显性解释色彩与构图的因果链，避免像两个话题拼接；
5. 第11章只处理输入、分析、理解、参考库，第12章只处理个人命题、项目执行、编辑输出；
6. 所有按钮、菜单、参数、后期工具都要回到画面判断，不写成说明书或软件教程；
7. 真实样张、拍摄参数、前后对比和作品案例优先；如果使用生成图，只能作为模拟教学图或示意图，并明确标注；
8. 每篇文章都要脱离当前写作上下文独立成立，避免使用依赖临时语境的表述。

## 0.7 验收标准

这套教程后续扩写时，应持续检查以下标准：

1. 学习者能理解色彩和构图不是并列知识，而是同一条注意力控制逻辑；
2. 全教程的练习都包含感知训练，不只停留在技术操作；
3. 第11章和第12章边界清晰：前者训练如何看，后者训练如何创作；
4. 每章至少有一个可执行、可比较、可复盘的练习；
5. 整套教程始终围绕观察、判断、拍摄、后期、复盘展开；
6. 教程覆盖从零基础到高阶创作，但不变成器材百科、参数百科或软件按钮索引。
