# 自定义按钮与C1C2C3

![案例图：自定义按钮和 C 模式服务不同拍摄场景](<./imgs/3.7-custom-buttons-c-modes.png>)

这组图把 AF-ON、摇杆、拨轮、自定义模式和不同场景放在一起。自定义不是把相机改得很复杂，而是把高频判断放到手指能立刻触达的位置，把一整套场景设置保存成可返回的起点。

## 1 这篇先解决什么画面问题

当你开始熟悉相机后，会遇到新的问题：

1. 常用功能藏在菜单里，现场找起来太慢；
2. 人像、街拍、风景、运动之间切换时，设置经常遗留；
3. 临时改了对焦、驱动或快门模式，下一次拍摄忘记改回；
4. 想使用后键对焦、对焦锁定、曝光锁定，却不知道是否适合自己；
5. C1/C2/C3 被保存成一堆参数，但没有对应场景和画面目标；

自定义的目标不是“让相机像高手的相机”，而是让你的常用判断更快、更稳定、更少遗留。

## 2 先观察：你真正高频的判断是什么

自定义前先做一周记录，看看你最常需要快速处理什么。

可以记录：

1. 是否经常切换 AF 区域；
2. 是否经常在人像眼控和普通单点之间切换；
3. 是否经常做曝光补偿；
4. 是否经常切换单拍和连拍；
5. 是否经常回放放大检查焦点；
6. 是否经常拍三脚架风景、手持街拍、室内人像等固定场景；

如果一个功能一周只用一次，不一定值得放到最顺手的按钮上。自定义应该从真实频率出发。

## 3 自定义按钮：把一个判断放到一个动作上

按钮自定义适合解决单个高频动作。例如：

1. AF-ON：启动自动对焦，适合后键对焦或临时追踪；
2. 摇杆：直接移动对焦点，适合离中心构图；
3. 星号或其他按键：曝光锁定、切换 AF 区域、眼部识别开关；
4. 镜头控制环：曝光补偿、光圈或 ISO，但要防止误触；
5. 回放相关按钮：快速放大检查焦点；

选择时遵守两个原则：

1. 重要功能要容易按，但不能容易误触；
2. 同一类判断尽量放在相近手指区域，减少记忆负担；

如果某个自定义让你拍摄时更慌，它就不是好自定义。

## 4 后键对焦：适合等待和追踪，但需要训练

后键对焦把对焦启动从快门半按里分离出来，常见做法是用 AF-ON 控制对焦，用快门按钮负责拍摄。

它的优势：

1. 等待时机时可以先完成对焦，再不反复重新对焦；
2. 追踪主体时可以持续按住 AF-ON；
3. 重新构图时手指分工更清楚；

它的代价：

1. 初学阶段容易忘记按 AF-ON；
2. 交给别人使用你的相机时，对方可能不会拍；
3. 如果没有形成肌肉记忆，会比默认半按更慢；

建议先用默认方式稳定拍摄，再用一周专门练后键对焦。不要在重要拍摄当天第一次启用。

## 5 C1/C2/C3：保存的是场景起点，不是万能模式

C1/C2/C3 适合保存一整套场景设置。它们的价值在于快速回到一个可靠起点。

可以这样设计：

| 模式 | 适合场景 | 可能保存的思路 |
| --- | --- | --- |
| C1 | 日常手持 | Av 或 M、Auto ISO、RAW+JPEG、单拍/低速连拍、常用 AF 区域； |
| C2 | 人像/人物走动 | Servo、人眼识别、适中连拍、适合肤色判断的白平衡策略； |
| C3 | 风景/三脚架 | 低 ISO、单拍或定时、较小光圈、防抖和快门方式按三脚架状态设置； |

表里的内容不是必须照抄。真正要保存的是：这个场景最怕什么失败。例如街拍最怕错过时机，风景最怕高光和稳定，室内人像最怕肤色和焦点。

## 6 自定义的更新规则

自定义不是一次设置终身不变。建议按这个节奏更新：

1. 新机前两周尽量少改，先理解默认逻辑；
2. 发现同一功能反复需要找菜单时，再考虑自定义；
3. 每次只改一个按钮，至少使用一周；
4. C 模式只在稳定场景形成后保存，不要每天乱存；
5. 更新后写一条摄影日志，记录为什么这样改；

如果自定义越来越多，但照片失败原因没有减少，就说明你在用配置逃避观察。

## 7 练习任务：建立一个可解释的 C1 模式

选择你最常拍的场景，例如日常记录。

### 7.1 操作步骤

1. 设定拍摄目标：快速记录、主体清楚、后期有余地；
2. 选择拍摄模式、画质、对焦、驱动、白平衡和回放显示；
3. 把这一套保存到 C1；
4. 改乱几个设置，再切回 C1，确认能回到起点；
5. 拍摄 20 张练习照片，观察是否减少遗留设置；

### 7.2 复盘问题

1. C1 解决了哪个真实场景；
2. 哪个设置最容易遗留，是否被 C1 修正；
3. 有没有保存太多不确定设置；
4. 是否需要 C2/C3，还是一个 C1 暂时够用；

## 8 本篇完成标准

完成这一篇后，你应该能做到：

1. 说出每个自定义按钮对应的拍摄判断；
2. 判断自己是否适合启用后键对焦；
3. 为至少一个 C 模式写出场景目标；
4. 避免为了复杂而复杂地改相机；
