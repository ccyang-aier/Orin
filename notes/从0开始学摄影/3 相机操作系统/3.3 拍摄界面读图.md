# 拍摄界面读图

![案例图：取景器和屏幕信息的读图方式](<./imgs/3.3-viewfinder-screen-readout.png>)

这组图用取景界面、曝光刻度、对焦框、直方图、高光提示和回放放大说明一件事：屏幕里的图标不是装饰，它们在告诉你照片可能会怎样失败。读界面不是背图标，而是把它们翻译成画面风险。

## 1 这篇先解决什么画面问题

很多新手会看取景器里的画面，却不读取景器里的状态。结果常见问题是：

1. 主体很好，但曝光补偿遗留在负值，整组照片偏暗；
2. 对焦框落在背景，人物眼睛其实不清楚；
3. 连拍没有打开，运动关键瞬间只拍到一张；
4. 白平衡或 Picture Style 遗留，颜色和预览误导判断；
5. 拍完只看缩略图，以为清楚，导入电脑才发现焦点错了；

拍摄界面是现场复盘工具。它让你在按快门前发现风险，在按快门后确认结果。

## 2 先观察：屏幕信息要服务哪五个问题

读界面时优先看五件事：

1. 亮度是否符合意图：主体、背景、高光和暗部是否在可接受范围；
2. 焦点是否落在真正主体：对焦框、眼部识别、追踪框是否可靠；
3. 时机是否能被记录：驱动模式和快门速度是否匹配运动；
4. 文件是否可用：画质、存储卡、电池和剩余张数是否安全；
5. 回看是否验证：放大、直方图、高光提示和对焦点显示是否帮助判断；

不要试图一次读完所有图标。先读会毁掉照片的项目。

## 3 曝光信息：亮度读数不是画面意图

界面里的曝光刻度、快门、光圈、ISO 和曝光补偿共同说明当前相机准备怎样记录光线。

读曝光时注意：

1. 曝光刻度靠中间，不等于照片一定正确；
2. 负曝光补偿可能保护高光，也可能让主体变黑；
3. ISO 升高可能保住快门速度，也可能让暗部噪点更明显；
4. 快门速度太慢，静物可能没问题，人物和手持可能失败；
5. 光圈太大，背景虚化更多，但多人或近距离静物可能焦平面不够；

屏幕预览如果开启曝光模拟，能帮助你直接看到变亮或变暗的趋势。但在闪光、强逆光或特殊显示设置下，预览也可能不是最终结果。关键是用界面信息和画面观察互相校验。

## 4 对焦信息：框在主体上，不等于主体一定清楚

对焦框、追踪框和眼部识别框只说明相机当前认为哪里重要。你还要判断它是否真的符合拍摄目标。

常见检查：

1. 人像：框是否落在更靠近相机的眼睛，而不是眉毛、头发或背景；
2. 动物：框是否稳定跟随眼睛或头部，而不是跳到身体或树枝；
3. 车辆和运动：框是否持续包住主体，而不是被背景高反差线条吸走；
4. 静物：框是否落在最需要清楚的质感、边缘或品牌信息上；
5. 多主体：相机是否选错人物，是否需要缩小 AF 区域或移动对焦点；

对焦界面越智能，越需要拍摄者保持判断。主体识别很有用，但它不是构图和表达的替代品。

## 5 直方图和高光提示：检查照片还能不能修

直方图不是只给后期人员看的。它能提醒你明暗信息是否挤在一起、是否明显贴边。

读直方图时先看：

1. 右侧是否大量贴边：可能高光溢出，例如天空、灯箱、婚纱、白墙；
2. 左侧是否大量贴边：可能暗部堵死，也可能是低调画面的正常选择；
3. 分布是否和画面意图一致：雪景偏右、夜景偏左都可能合理；
4. RGB 直方图是否出现某个颜色通道溢出，尤其是红色花朵、舞台灯、霓虹；

高光警告适合检查不可恢复的亮区。它不代表所有闪烁区域都必须救回，而是提醒你：这里如果没有细节，是否会破坏这张照片。

## 6 驱动、画质、电池和卡：保证照片能被记录

界面底部和边缘的信息常常被忽略，但它们会影响拍摄是否成功：

1. 驱动模式：单拍、低速连拍、高速连拍、自拍定时会改变记录时机；
2. 画质格式：RAW、JPEG 或 RAW+JPEG 会影响后期空间和文件大小；
3. 白平衡：预览颜色是否稳定，是否误导肤色和环境判断；
4. 存储卡：当前记录卡、剩余张数和写入状态是否安全；
5. 电池：长时间拍摄前是否还能支撑后续流程；

这些信息不一定每张都要检查，但在重要拍摄前必须确认一次。

## 7 回放界面：拍完才是第一次小复盘

回放不要只看构图和表情。至少检查四件事：

1. 放大到主体关键处，看眼睛、产品边缘、纹理或动作关键点是否清楚；
2. 查看高光提示，判断亮区是否需要重拍；
3. 看边缘有没有突兀亮点、半截人物、杂物或切断线条；
4. 如果机身支持对焦点显示，查看实际 AF 点和主体位置是否一致；

现场回放的目标不是沉迷选片，而是及时发现可以重拍的问题。能当场修正的错误，不要留给后期。

## 8 练习任务：读界面后再按快门

选择一个窗边静物或人物，连续拍摄四张，每次按快门前都读一次界面。

### 8.1 拍摄步骤

1. 第一张：正常曝光，记录当前界面状态；
2. 第二张：故意降低曝光补偿，观察取景器和直方图变化；
3. 第三张：移动对焦点到主体关键位置；
4. 第四张：改变驱动或快门速度，拍一个轻微动作；

### 8.2 复盘问题

1. 哪个界面信息最能提前提醒失败；
2. 直方图和你眼睛看到的亮度是否一致；
3. 对焦框有没有骗过你；
4. 回放放大后，哪一张看似成功但实际不够清楚；

## 9 本篇完成标准

完成这一篇后，你应该能做到：

1. 从取景界面判断曝光、对焦和驱动是否适合当前画面；
2. 用直方图和高光提示判断是否需要重拍；
3. 用回放放大检查真正清晰度；
4. 不把界面图标当成装饰，也不让图标替代画面判断；
