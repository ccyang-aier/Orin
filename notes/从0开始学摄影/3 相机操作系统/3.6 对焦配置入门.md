# 对焦配置入门

![案例图：不同主体和场景下的自动对焦配置](<./imgs/3.6-autofocus-configuration-cases.png>)

这组图展示了人像眼部、骑行运动、鸟类、车辆、静物和失败对焦几种情况。对焦配置的核心不是让相机永远自动找对，而是让相机知道你要它优先服务哪个主体，以及你是否愿意让它持续追踪。

## 1 这篇先解决什么画面问题

对焦失败通常不是“相机不行”这么简单。常见问题包括：

1. 焦点落在背景，主体脸部是虚的；
2. 人像眼部识别跳到另一张脸或头发边缘；
3. 运动主体进入画面后，AF 区域太小导致追丢；
4. 静物拍摄时 AF 区域太大，焦点落到高反差背景；
5. 多人合影只对到前排一张脸，其他人不清楚；
6. 误把 Servo、One Shot、主体识别和 AF 区域混成一个概念；

对焦配置要解决的是：让清楚的区域和你的画面重点一致。

## 2 先观察：主体是否稳定、明确、可识别

选择对焦方式前，先判断主体状态：

1. 主体是否静止：静物、风景、摆拍人像通常更稳定；
2. 主体是否移动：儿童、运动、动物、街拍人物需要持续追踪；
3. 主体是否容易被识别：人脸、人眼、动物、车辆通常适合主体识别；
4. 背景是否容易抢焦：树枝、栏杆、文字、灯牌、高反差边缘都可能误导；
5. 构图是否稳定：主体是否从中央移动到边缘，是否需要移动对焦点；

对焦不是相机内部事件，它从观察主体开始。

## 3 AF 区域：你给相机多大的选择范围

AF 区域越小，拍摄者控制越强，但需要更准确地放置；AF 区域越大，相机选择空间越大，但也更可能选错。

可以这样理解：

1. 单点或小区域：适合静物、细节、遮挡较多的人像、背景复杂场景；
2. 区域 AF：适合主体位置有变化但仍在某个范围内，例如人物走动；
3. 全区域追踪：适合主体识别明显、移动路径较大、需要相机辅助跟随；
4. 触控选择主体：适合屏幕取景时直接告诉相机从哪里开始追踪；

如果背景很干净，大区域和追踪通常很好用；如果背景很复杂，缩小区域反而更稳定。

## 4 主体识别：它会找人、动物或车辆，但不替你决定主题

现代相机的主体识别可以识别人、动物、车辆等对象，并尝试锁定脸部、眼部或主体轮廓。它非常有用，但它仍然需要拍摄者给出边界。

人像时要看：

1. 框是否落在更靠近相机的眼睛；
2. 多人场景是否选中了真正主体；
3. 侧脸、背光、遮挡和低反差下是否稳定；

动物和车辆时要看：

1. 识别是否持续，不要只在主体清楚时才出现；
2. 背景是否比主体更容易被相机抓住；
3. 主体进入画面的方向是否需要预留空间；

主体识别能提高效率，但不能替你决定哪张脸、哪只动物或哪辆车是照片的重点。

## 5 One Shot 与 Servo：一次确认还是持续追踪

One Shot 适合焦距关系稳定的场景。半按或 AF-ON 后，相机会完成一次对焦，适合静物、摆拍人像、建筑、风景和慢节奏构图。

Servo 适合主体和相机距离持续变化的场景。相机会持续调整焦点，适合运动、人像走动、儿童、动物、车辆、街拍跟随。

选择时问：

1. 主体到相机的距离是否在变化；
2. 我是否需要按下快门前继续追踪；
3. 表情或动作是否需要等待；
4. 如果锁定一次焦点，下一秒主体是否已经离开焦平面；

如果主体静止，用 Servo 也可以拍，但可能增加不必要的不确定；如果主体运动，用 One Shot 锁一次焦点，往往会在真正按下快门时失焦。

## 6 追踪灵敏度：别让相机太容易放弃主体

追踪灵敏度、加减速跟踪等设置属于更进阶的 AF 行为控制。新手不必先背所有参数，但要理解它们服务的画面问题：

1. 主体短暂被遮挡时，是否希望相机继续相信原主体；
2. 新对象突然进入画面时，是否希望相机迅速切过去；
3. 主体速度忽快忽慢时，是否需要更积极预测；

街拍中有人从主体前经过，你可能不希望相机立刻跳到路人；球类运动中目标快速变向，你可能希望相机更积极追踪。这里没有万能设置，只有场景取舍。

## 7 对焦失败的诊断流程

一张照片不清楚时，按顺序判断：

1. 是手抖还是主体运动：整张都虚，还是只有运动方向拖影；
2. 是景深不够还是焦点落错：前后某处是否有清楚区域；
3. 是 AF 区域太大还是太小：相机有没有机会选错；
4. 是 One Shot/Servo 用错：主体是否在按快门前移动；
5. 是快门速度不够：对焦正确但动作仍然糊；
6. 是回放检查不足：现场是否已经能发现问题；

对焦失败常常和快门速度、景深、站位一起出现。不要只怪 AF。

## 8 练习任务：同一主体做三种对焦

找一个愿意配合的人，或者用一个静物和一个移动物体完成练习。

### 8.1 拍摄步骤

1. 静物：使用小 AF 区域，对准最重要纹理拍一张；
2. 人像：使用人眼识别，拍正脸、侧脸和轻微遮挡各一张；
3. 运动：使用 Servo 和较大区域，拍人物走向镜头或骑车经过；
4. 失败样张：故意让背景更亮或更清晰，观察相机是否误吸焦；

### 8.2 复盘问题

1. 哪种场景小区域更稳；
2. 哪种场景主体识别明显提高成功率；
3. 哪张照片焦点落错，原因是什么；
4. 失焦和快门速度不足能否区分；

## 9 本篇完成标准

完成这一篇后，你应该能做到：

1. 根据主体运动状态选择 One Shot 或 Servo；
2. 根据背景复杂度选择 AF 区域大小；
3. 使用主体识别时仍然确认真正主体；
4. 用回放放大诊断对焦失败原因；
