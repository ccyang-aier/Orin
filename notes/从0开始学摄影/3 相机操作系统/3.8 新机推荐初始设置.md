# 新机推荐初始设置

![案例图：新手稳妥初始设置服务不同拍摄场景](<./imgs/3.8-beginner-initial-settings.png>)

这组图把初始设置和真实画面连接起来：RAW+JPEG 保护后期空间，白平衡影响肤色，驱动和对焦影响运动，低 ISO 和回放检查影响风景与静物质量。推荐设置不是终点，只是让新手从一个不容易出错的起点开始练习。

## 1 这篇先解决什么画面问题

新机到手时，最危险的不是不会把所有功能打开，而是太快进入复杂配置。

常见失败包括：

1. 照抄网上设置，自己不知道每一项影响什么；
2. 打开很多高级功能，结果现场不知道哪里出错；
3. 只拍 JPEG，练习曝光和后期时没有余地；
4. 对焦模式、驱动模式和快门类型被上一次拍摄遗留；
5. 过早追求“直出色彩”，忽略主体、光线和背景；

初始设置的目标是稳定、可复盘、低风险。它不负责让照片立刻高级。

## 2 初始设置原则：少改、可见、可恢复

新手机身可以遵守三个原则：

1. 少改：只改会明显影响拍摄和安全的项目；
2. 可见：让取景器和回放界面显示足够信息，方便发现失败；
3. 可恢复：用 C 模式或记录表保留一个稳定起点；

不要一上来就改复杂 AF Case、自定义曲线、色彩风格、深层视频格式、HDR PQ、Dual Pixel RAW 等项目。等你能说清楚问题再改。

## 3 推荐初始设置

以下是适合新手练习的稳妥起点。不同品牌菜单名称不同，但逻辑相通。

| 设置项 | 推荐起点 | 画面理由 |
| --- | --- | --- |
| 画质 | RAW+JPEG | JPEG 方便快速查看，RAW 保留后期和曝光复盘空间； |
| 测光 | 评价/矩阵测光 | 大多数场景稳定，再用曝光补偿表达意图； |
| 白平衡 | 自动白平衡 | 先减少操作负担，重要系列再固定色温或预设； |
| 对焦 | 人像用主体/眼部识别，静物用小区域 | 让主体清楚，同时避免复杂背景误吸焦； |
| AF 操作 | 静态用 One Shot，运动用 Servo | 根据主体距离是否变化选择； |
| 驱动 | 单拍起步，动作场景用低速或高速连拍 | 避免无意义连拍，也不漏掉关键瞬间； |
| 快门类型 | 机械或电子前帘优先 | 比全电子快门更稳妥，减少运动变形和频闪风险； |
| 防抖 | 手持开启，三脚架按说明和现场测试决定 | 防抖帮助手持稳定，但不凝固主体运动； |
| 无卡释放 | 禁用 | 避免看似拍到但没有文件； |
| 回放显示 | 开启直方图/高光提示/放大检查路径 | 让现场复盘有证据； |

这不是唯一答案，但它能让新手把主要精力放回画面。

## 4 暂时不要急着改的设置

有些设置很吸引人，但不适合一开始就大幅调整：

1. Picture Style：会影响 JPEG 和预览观感，但不能替代光线和曝光判断；
2. 色彩空间：网络分享通常先用 sRGB，等进入打印或专业输出再系统学习；
3. 高级 AF Case：先理解主体、区域、One Shot 和 Servo，再调追踪性格；
4. HDR、HDR PQ 或特殊高动态模式：先学会普通曝光取舍；
5. Dual Pixel RAW 或特殊 RAW 功能：文件更大，收益取决于后期流程；
6. 复杂自定义按钮：先知道默认按钮逻辑，再按真实高频需求修改；
7. 视频专业格式：除非本阶段目标是视频，否则不要让它干扰静态摄影学习；

越高级的设置，越需要明确问题。问题还不清楚时，保守设置更适合学习。

## 5 三个场景的起点

### 5.1 日常记录

目标是快速、稳定、后期有余地：

1. RAW+JPEG；
2. Av 或 P 起步，Auto ISO；
3. 评价测光，曝光补偿用拨轮调整；
4. 自动白平衡；
5. 主体识别按场景启用；
6. 单拍或低速连拍；

### 5.2 人像练习

目标是眼睛清楚、肤色可信、背景不抢：

1. RAW+JPEG；
2. Av 起步，按景深需要调整光圈；
3. 人眼识别或小区域 AF；
4. 静态用 One Shot，走动用 Servo；
5. 注意白平衡和环境反光；
6. 回放放大检查眼睛；

### 5.3 风景和静物

目标是稳定、细节、影调可控：

1. RAW；
2. 低 ISO；
3. 单拍或定时；
4. 按三脚架和镜头状态判断防抖；
5. 使用直方图检查高光；
6. 必要时包围曝光，但先明确保留什么；

## 6 初始设置记录表

建议在摄影日志中写一张表：

| 项目 | 当前设置 | 为什么这样设 | 什么时候改 |
| --- | --- | --- | --- |
| 画质 | RAW+JPEG | 方便练习和复盘； | 文件管理稳定后可按项目调整； |
| AF | 依据主体切换 | 避免一套 AF 拍所有场景； | 进入运动、动物、街拍时细分； |
| 驱动 | 单拍起步 | 先控制每一次判断； | 动作和表情需要时切连拍； |
| 回放 | 直方图/高光/放大 | 现场发现失败； | 熟练后按显示习惯简化； |

记录比记忆可靠。每一次修改都应该能写出原因。

## 7 练习任务：建立自己的默认起点

拿到相机后，不要先外出拍大量照片。先在一个稳定场景完成默认起点测试。

### 7.1 操作步骤

1. 按推荐表完成基础设置；
2. 拍一个静物、一个人像、一个轻微动作；
3. 每张都回放放大检查焦点；
4. 查看直方图和高光提示；
5. 写下哪项设置让你最安心，哪项设置还不理解；

### 7.2 合格标准

1. 三类照片都被成功记录；
2. 没有无卡、画质错误、对焦区域明显错误；
3. 能根据主体运动选择 One Shot 或 Servo；
4. 能说明下一次最需要练习的设置；

### 7.3 复盘问题

1. 哪项设置最容易遗留；
2. 哪项设置你是照抄但还没理解；
3. 是否需要保存一个 C1 起点；
4. 哪些高级功能可以暂时不碰；

## 8 本篇完成标准

完成这一篇后，你应该能做到：

1. 建立一套适合新手练习的稳妥默认值；
2. 知道每个默认值对应的画面或流程理由；
3. 区分“现在必须设置”和“以后再学也不迟”；
4. 不让复杂菜单抢走观察、光线和构图训练；
