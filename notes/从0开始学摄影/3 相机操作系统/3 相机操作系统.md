# 相机操作系统

第 3 篇不是说明书复述，而是帮助学习者把相机变成稳定的拍摄工具。学习相机操作时，最容易走偏的地方是把按钮、菜单和设置当成孤立知识：知道某个选项在哪里，却不知道它应该在什么画面问题里被调用。

本篇的核心目标是建立一套操作顺序：

1. 拿起相机前先排除会中断拍摄的风险；
2. 用按钮和拨轮把曝光、对焦、回看和菜单变成手上的动作；
3. 在取景器和屏幕里读懂画面状态，而不是只看图标；
4. 把 Q 快速菜单、主菜单和 My Menu 分配到不同决策层级；
5. 先设置会影响照片结果的核心项目，再处理个性化选项；
6. 让对焦、按钮自定义和 C1/C2/C3 服务真实场景；

正文以 Canon EOS R6 Mark II 作为示例，是因为具体机型能让按钮和菜单落地。学习时不要只记住 Canon 的路径，而要把每一项翻译成通用问题：这项设置是否影响画质、曝光、对焦、快门时机、稳定性、回看判断或文件安全。

## 2 本篇完成标准

完成本篇后，学习者不需要背下所有菜单，但应该能做到：

1. 拍摄前独立完成电池、卡、镜头、肩带、时间、画质和文件安全检查；
2. 不离开取景器完成曝光补偿、对焦点移动、回放放大和常用设置确认；
3. 看到取景界面时能判断当前照片最可能失败在哪里；
4. 知道哪些设置应该放在 Q，哪些应该进入 My Menu，哪些只需要低频检查；
5. 能根据静物、人像、运动、风景和街拍切换基本对焦策略；
6. 能说清楚自己的自定义按钮和 C 模式为什么这样配置；

## 3 本篇练习总任务

选择一个简单场景，例如窗边人物、桌面静物或街角行人。完成以下操作，不以照片好看为第一目标，而以操作稳定为目标：

1. 开拍前检查电池、存储卡、镜头、时间、画质和防抖；
2. 不离开取景器完成一次曝光补偿；
3. 切换一次对焦区域，并让对焦框落在真正主体上；
4. 切换一次驱动模式，用单拍和连拍分别拍一组；
5. 回放放大检查清晰度，并查看高光是否明显溢出；
6. 把一个常用设置加入 My Menu，并说明为什么它值得放进去；

复盘时不要只写“操作熟不熟”。要写出每一步对应的画面风险：曝光补偿解决什么亮度问题，对焦区域解决什么主体问题，驱动模式解决什么时机问题，回放检查解决什么失败成本。
