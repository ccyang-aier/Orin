# 拿起相机前

![案例图：开拍前的相机准备检查](<./imgs/3.1-camera-preparation-checklist.png>)

这组图把一次拍摄前最容易被忽略的准备动作放在一起：确认电池余量、插好存储卡、装好镜头、固定肩带、检查基础信息，最后让相机进入可以随时举起拍摄的状态。它不是器材仪式，而是在保护后面的观察、判断和时机。

## 1 这篇先解决什么画面问题

很多照片失败，不是发生在按快门时，而是发生在按快门之前。

常见情况包括：

1. 光线刚好出现，电池却已经没电；
2. 人物表情到位，存储卡却满了或没有插卡；
3. 拍到一半发现镜头前后盖、滤镜、转接环或防抖状态没有检查；
4. 长时间拍摄后发现相机时间不准，照片排序和多机位整理混乱；
5. 拍摄现场临时翻菜单，注意力从画面转移到机器；

这些问题看起来不像摄影审美，但它们会直接毁掉画面。摄影中的准备不是为了显得专业，而是为了让真正的画面判断不被工具打断。

## 2 先观察：这次拍摄最怕什么中断

开拍前先想清楚这次拍摄的失败成本。不同场景的准备重点不同：

1. 日常练习：最怕卡没带、电池不够、画质格式不对，导致练习样张不能复盘；
2. 人像拍摄：最怕拍到一半断电、卡满、时间混乱，打断被拍者状态；
3. 风景拍摄：最怕到达光线窗口时三脚架、快门线、滤镜、卡和电池没有准备好；
4. 街拍和旅行：最怕相机不在随时可拍状态，错过短暂关系；
5. 作品项目：最怕文件命名、日期和版权信息混乱，后期无法追踪；

准备动作要围绕这次拍摄最脆弱的环节，而不是机械地检查所有东西。

## 3 电池：余量不是数字，而是拍摄节奏

电池检查要回答三个问题：

1. 当前电量是否足够完成这次拍摄；
2. 是否有备用电池，并且备用电池已经充满；
3. 拍摄环境是否会让电量消耗变快，例如低温、长时间开屏、视频、连拍、无线传输；

如果是一次普通练习，可以接受一块电池边拍边观察。但如果是不可重复的场景，例如日出、活动、人像约拍或旅行当日行程，备用电池就不是附件，而是拍摄机会的保险。

拍摄前把电池逻辑写成一句话：

> 这次拍摄如果持续两小时，我是否还能在最后十分钟保持相机可用？

## 4 存储卡：卡槽不是仓库，是临时容器

存储卡检查不只是“有没有卡”。至少要看四件事：

1. 卡是否插入正确卡槽；
2. 卡剩余容量是否足够；
3. 卡内旧文件是否已经备份，不会被误删；
4. 当前画质设置下，卡的写入能力是否能支撑连拍或视频；

以 Canon EOS R6 Mark II 这类双卡槽机身为例，可以把双卡理解成两种安全策略：一张卡记录 RAW，另一张卡记录 JPEG；或者两张卡同时备份同一张照片。对新手来说，重要的不是一开始就设计复杂存储方案，而是避免“只拍在一张快满的卡上，还没有备份”的高风险状态。

拍摄前最低规则：

1. 不在未备份的卡上随意格式化；
2. 不把所有重要照片长期留在卡里；
3. 卡满之前就更换，不等相机提示才处理；
4. 清卡只在确认导入和备份之后进行；

## 5 镜头、机身和肩带：先排除物理风险

镜头检查要看画面，也要看安全：

1. 镜头是否正确锁定在卡口上；
2. 前后镜片是否有明显灰尘、指纹、水渍或雾气；
3. 镜头开关是否在需要的位置，例如 AF/MF、防抖、焦段锁；
4. 滤镜是否影响画面，例如脏 CPL、错误 ND、廉价保护镜反光；
5. 机身和镜头之间是否有转接环，转接环是否锁紧；

肩带和快装板也会影响拍摄。松动的肩带可能让相机跌落；松动的快装板可能让三脚架构图在慢门中轻微位移。稳定不是只属于快门速度，也属于器材和身体的连接方式。

## 6 固件、时间和版权信息：让文件在未来可用

固件不需要每次拍摄前都更新，但需要定期知道自己使用的版本。固件可能影响对焦、镜头兼容、功能稳定性或错误修复。重要拍摄前不要临时更新固件，除非你已经确认新版本稳定并完成试拍。

时间设置影响文件排序、跨设备同步和摄影日志。多台相机或手机混拍时，时间不准会让后期选片非常痛苦。出发前确认：

1. 日期、时间和时区正确；
2. 夏令时或旅行时区没有误设；
3. 文件编号和文件夹逻辑不会打乱当前项目；

版权信息不是防盗魔法，但它能把基础作者信息写入文件元数据。对于长期项目、交付照片和作品归档，这是一种低成本的身份标记。不要把它当成法律保护的全部，只把它当作文件管理的一部分。

## 7 开拍前最小检查清单

真正到现场时，检查清单要短，否则会把注意力从画面抽走。

可以使用这 10 项：

1. 电池：当前电量足够，备用电池可用；
2. 存储卡：已插卡，容量足够，旧文件已备份；
3. 画质：RAW、JPEG 或 RAW+JPEG 符合本次目标；
4. 镜头：安装锁定，镜片干净，AF 和防抖状态正确；
5. 时间：日期、时间和时区正确；
6. 肩带：固定可靠，不影响举机；
7. 模式：当前拍摄模式不是上一次遗留的特殊模式；
8. 对焦：AF 区域和主体识别适合当前场景；
9. 曝光：ISO、曝光补偿和测光没有遗留异常；
10. 回看：能放大检查清晰度，能看到高光或直方图信息；

## 8 练习任务：三分钟开拍前检查

选择一个简单场景，例如窗边静物或家人坐在窗边。不要急着拍，先完成一次三分钟检查。

### 8.1 操作步骤

1. 把相机放在桌上，从关机状态开始；
2. 装好电池、卡和镜头；
3. 检查画质、时间、对焦、曝光补偿和防抖；
4. 举起相机拍一张测试图；
5. 回放放大检查清晰度，并确认文件已经记录；

### 8.2 合格标准

1. 三分钟内完成准备，没有遗漏卡、电池、镜头和画质；
2. 测试照片主体清楚，曝光没有明显异常；
3. 回放能确认照片已经记录到卡上；
4. 能说出本次最怕的一个中断风险；

### 8.3 复盘问题

1. 哪一步最容易忘；
2. 哪个遗留设置最可能毁掉下一张照片；
3. 如果是人像、风景、街拍或旅行，这张检查表需要增加什么；
4. 你的相机包里是否有重复、缺失或长期不用的物品；

## 9 本篇完成标准

完成这一篇后，你应该能做到：

1. 不依赖说明书完成一次开拍前检查；
2. 知道电池、卡、镜头、时间和画质分别会怎样影响照片；
3. 在重要拍摄前避免临时更新和临时改复杂设置；
4. 把器材准备理解为保护画面判断，而不是拍摄前的形式感；
