# 机身按钮地图

![案例图：机身按钮和拨轮的操作关系](<./imgs/3.2-camera-button-map.png>)

这组图用特写展示相机上最常被手指触碰的控制点：快门、主拨轮、速控拨轮、方向摇杆、AF-ON、Q、MENU、回放和录像相关按钮。学习按钮的关键不是记住位置，而是知道每个按钮解决哪一类画面问题。

## 1 这篇先解决什么画面问题

新手拿起相机时，经常出现两种极端：

1. 只按快门，其他按钮都不敢碰；
2. 到处翻菜单，现场真正的光线、主体和时机反而被忽略；

按钮地图要解决的是“拍摄判断到手指动作之间的延迟”。你看到画面偏亮，手指能不能立刻做曝光补偿；你发现主体从画面中央移动到边缘，能不能移动对焦点；你拍完不确定清晰度，能不能马上回放放大。

如果每一个判断都需要低头找按钮，照片就会慢半拍。相机熟练不是炫技，而是让观察不被操作打断。

## 2 先观察：哪类判断最需要即时操作

拍摄现场最常见的即时判断有五类：

1. 曝光判断：主体太暗、背景太亮、高光快溢出，需要立刻调整曝光补偿、光圈、快门或 ISO；
2. 对焦判断：主体移动、构图改变、对焦点落到背景，需要立刻移动 AF 框或切换区域；
3. 时机判断：人物动作开始、表情出现、运动进入关键位置，需要立刻切换驱动或连拍；
4. 检查判断：拍完要确认清晰度、表情、高光和边缘，不是只看缩略图；
5. 设置判断：少量常用项目要在 Q 或 My Menu 中迅速找到；

按钮地图应该围绕这些判断建立，而不是围绕按钮本身建立。

## 3 快门和半按：从观察进入拍摄

快门按钮不是只有“按下去”一个动作。半按通常承担测光、对焦或准备拍摄的角色，完全按下才是记录。

使用时要注意：

1. 半按可以让相机进入准备状态，但不要用半按替代观察；
2. 静态主体可以半按确认对焦，再重新构图，但大幅移动构图可能导致焦点偏移；
3. 动态主体更适合持续追踪或 Servo，而不是半按锁死一次焦点；
4. 如果使用后键对焦，快门按钮和 AF-ON 的分工会改变，需要重新训练手指；

判断标准不是“哪种按法更高级”，而是当前主体是否稳定，焦点是否会在你构图变化中丢失。

## 4 主拨轮和速控拨轮：把曝光变成身体动作

拨轮最重要的作用是让曝光调整不用离开取景器。

不同模式下，拨轮控制的项目可能不同。例如在 Av/A 模式下常用于调整光圈和曝光补偿；在 Tv/S 模式下常用于调整快门速度和曝光补偿；在 M 模式下常用于调整光圈、快门或 ISO 的组合。

学习时不要先背“哪个拨轮一定控制什么”，而要建立三个动作：

1. 画面太亮或高光刺眼：快速降低曝光或做负曝光补偿；
2. 背景太清楚或太虚：快速调整光圈，改变景深；
3. 动作拖影或太凝固：快速调整快门速度；

熟练的标志是：你能一边看取景器里主体和背景的变化，一边用拨轮微调，而不是每调一次就低头确认。

## 5 方向摇杆和对焦点：让主体真的被看见

方向摇杆或多功能控制器是主体位置的控制入口。构图不是永远把主体放中间，对焦也不应该永远停在中间。

使用它时先问：

1. 主体现在位于画面哪里；
2. 对焦框是否落在主体最重要的位置，例如眼睛、脸、产品标识或动作关键点；
3. 背景里是否有更高反差、更清晰、更容易误吸焦的物体；
4. 当前 AF 区域是否太大，导致相机替你选择了错误对象；

当你能用摇杆移动对焦点，就不必为了对焦被迫把主体放回中央。构图自由度会增加，但同时你也要更主动地确认焦点位置。

## 6 AF-ON：把对焦从按快门里拆出来

AF-ON 常被用来启动自动对焦。很多进阶用户会把对焦交给 AF-ON，把快门按钮更多地用于拍摄记录，这通常被称为后键对焦。

但新手不必一开始就强行改成后键对焦。更好的学习顺序是：

1. 先理解半按快门对焦的默认逻辑；
2. 再尝试用 AF-ON 启动对焦，观察手指分工是否更稳定；
3. 最后根据自己的拍摄类型决定是否把对焦和快门分离；

后键对焦适合需要持续追踪、频繁重构图、等待时机的场景。它也会增加学习成本。如果你还在混淆曝光、焦点和构图，先保持默认设置也完全可以。

## 7 Q、MENU、回放和录像按钮：三种不同速度

这些按钮服务不同层级：

1. Q：现场频繁修改，速度最快，例如画质、驱动、白平衡、AF 区域、测光、触控等；
2. MENU：低频基础配置，速度较慢，例如格式化、版权、存储卡记录方式、自定义按钮；
3. 回放：拍完立即复盘，检查清晰度、表情、高光、构图边缘和对焦点；
4. 录像：拍视频时进入另一套判断，不要让误触改变当前拍摄状态；

把常用判断交给 Q，把低频设置留给 MENU，把拍完检查交给回放。这样相机操作会有清楚层级。

## 8 练习任务：不离开取景器完成四个动作

选择一个静物或人物，保持相机在眼前，完成以下练习。

### 8.1 拍摄步骤

1. 半按快门或 AF-ON 完成一次对焦；
2. 用拨轮做一次曝光补偿，让画面明显变暗或变亮；
3. 用摇杆移动对焦点，让主体从中央移到三分线附近；
4. 拍摄后回放放大，检查主体最重要位置是否清楚；

### 8.2 合格标准

1. 四个动作中途不进入深层菜单；
2. 曝光变化能在取景器里被看见；
3. 对焦点移动后主体仍然清楚；
4. 回放检查不是只看“拍到了”，而是能判断清晰度和高光；

### 8.3 复盘问题

1. 哪个按钮最容易找错；
2. 哪个动作最容易让你错过时机；
3. 你是否因为不会移动对焦点而改变了构图；
4. 下一次最应该训练哪一个手指动作；

## 9 本篇完成标准

完成这一篇后，你应该能做到：

1. 把主要按钮和拨轮对应到曝光、对焦、时机、回看和设置；
2. 不把 Q 和 MENU 混成同一个入口；
3. 用摇杆或控制器移动对焦位置；
4. 拍完后快速回放放大检查关键清晰度；
